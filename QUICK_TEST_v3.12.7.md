### 测试2：图标获取

#### 步骤
1. 打开编辑器
2. 点击"插入链接"按钮
3. 选择"卡片链接"
4. 输入以下网站链接
5. 点击"✨ 自动获取"按钮
6. 检查是否获取到图片

#### 测试用例

**用例1：有Open Graph图片的网站**
```
https://github.com
```
预期：✅ 获取到GitHub的og:image

**用例2：只有favicon的网站**
```
https://stackoverflow.com
```
预期：✅ 获取到Stack Overflow的favicon

**用例3：有Apple Touch Icon的网站**
```
https://developer.mozilla.org
```
预期：✅ 获取到MDN的图标

**用例4：只有默认favicon.ico的网站**
```
任何小型个人网站
```
预期：✅ 获取到/favicon.ico

#### 预期结果
- ✅ 至少85%的网站能获取到图标
- ✅ 图片URL显示在"卡片图片URL"输入框中
- ✅ 插入后卡片显示图片

---

### 测试3：删除功能

#### 步骤
1. 打开编辑器
2. 插入3-5个链接卡片
3. 将鼠标悬停在第一个卡片上
4. 观察右上角是否显示"编辑"和"删除"按钮
5. 点击"删除"按钮
6. 在确认对话框中点击"确定"

#### 预期结果
- ✅ 悬停时按钮淡入显示
- ✅ 点击删除按钮后弹出确认对话框
- ✅ 确认后卡片立即从文档中移除
- ✅ 显示"链接卡片已删除"的成功提示
- ✅ 其他卡片不受影响

#### 错误情况测试
如果删除失败：
- ✅ 应该显示错误提示："找不到要删除的卡片"
- ✅ 控制台应该输出错误日志
