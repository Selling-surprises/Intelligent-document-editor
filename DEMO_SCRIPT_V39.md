## 演示1：实时格式检测（2分钟）

### 步骤
1. **输入标题**
   ```
   输入："这是一个标题"
   选中文字 → 选择"标题 1"
   ```

2. **观察工具栏**
   ```
   点击标题文字
   👀 观察：标题选择器显示"标题 1"
   ```

3. **切换到正文**
   ```
   点击下方空白处
   输入："这是正文内容"
   👀 观察：标题选择器显示"正文"
   ```

4. **测试字体检测**
   ```
   选中正文
   选择字体："Microsoft YaHei"
   点击文字
   👀 观察：字体选择器显示"Microsoft YaHei"
   ```

5. **测试字号检测**
   ```
   选中正文
   选择字号："24px"
   点击文字
   👀 观察：字号选择器显示"24px"
   ```

### 演示要点
- ✨ 强调"实时"和"自动"
- ✨ 展示多次切换
- ✨ 说明提升了编辑体验

## 演示2：智能链接插入（2分钟）

### 步骤
1. **方式一：选中文字插入**
   ```
   输入："访问百度搜索引擎"
   选中"百度搜索引擎"
   点击链接按钮
   输入URL：https://www.baidu.com
   点击插入
   👀 观察：链接文本是"百度搜索引擎"
   ```

2. **方式二：手动输入文本**
   ```
   点击空白处
   点击链接按钮
   输入URL：https://www.google.com
   输入文本：谷歌搜索
   点击插入
   👀 观察：链接文本是"谷歌搜索"
   ```

3. **方式三：只输入URL**
   ```
   点击空白处
   点击链接按钮
   输入URL：https://github.com
   文本留空
   点击插入
   👀 观察：链接文本是URL本身
   ```

### 演示要点
- ✨ 强调"选中文字"最方便
- ✨ 说明三种方式的适用场景
- ✨ 展示成功提示消息

## 演示3：图片查看器（3分钟）

### 步骤
1. **插入图片**
   ```
   点击"插入图片"按钮
   上传2-3张图片
   ```

2. **导出HTML**
   ```
   点击右侧"导出HTML"按钮
   保存文件
   ```

3. **打开HTML文件**
   ```
   在浏览器中打开导出的HTML文件
   ```

4. **测试图片查看器**
   ```
   点击第一张图片
   👀 观察：图片全屏显示，黑色背景
   
   点击"+"按钮
   👀 观察：图片放大
   
   点击"−"按钮
   👀 观察：图片缩小
   
   点击"⟲"按钮
   👀 观察：图片重置
   
   按ESC键
   👀 观察：查看器关闭
   ```

5. **测试其他图片**
   ```
   点击第二张图片
   重复上述操作
   ```

### 演示要点
- ✨ 强调"导出的HTML文件"
- ✨ 展示流畅的动画效果
- ✨ 说明支持键盘操作
- ✨ 强调完全离线运行

## 演示话术

### 开场白
```
大家好！今天我要演示离线Word文档编辑器的v3.9版本。
这个版本带来了三个重要的功能改进，
让编辑体验更加流畅和直观。
```

### 功能1：实时格式检测
```
首先是实时格式检测功能。
以前我们不知道当前光标位置是什么格式，
现在工具栏会自动显示当前的标题级别、字体和字号。

（演示操作）

看，当我点击标题时，工具栏自动显示"标题 1"。
当我点击正文时，自动显示"正文"。
这让编辑变得更加直观。
```

### 功能2：智能链接插入
```
第二个功能是智能链接插入。
以前插入链接时，即使选中了文字，
也要手动输入链接文本，很不方便。

现在，我们优化了这个流程。

（演示操作）

看，我选中"百度搜索引擎"这几个字，
点击链接按钮，只需要输入URL，
它会自动使用选中的文字作为链接文本。

如果没有选中文字，也可以手动输入，
甚至可以留空，直接使用URL作为文本。

非常灵活！
```

### 功能3：图片查看器
```
第三个功能是图片查看器。
这是一个非常实用的功能。

以前导出的HTML文件中，图片只能静态显示，
无法放大查看细节。

现在，我们为导出的HTML添加了完整的图片查看器。

（演示操作）

看，我先插入几张图片，然后导出为HTML。
在浏览器中打开后，点击任意图片，
就会全屏显示，并且可以放大、缩小、重置。

按ESC键或点击背景就可以关闭。

这个功能完全不依赖任何外部库，
纯原生JavaScript实现，
可以在任何浏览器中运行。
```

### 结束语
```
以上就是v3.9版本的三个主要功能。

这些改进都是基于用户反馈实现的，
我们会持续优化，为大家提供更好的编辑体验。

感谢大家的支持！
```

## 常见问题应对

### Q1：格式检测不准确怎么办？
```
A：格式检测是基于光标位置的，
   确保光标在要检测的文字内部。
   如果检测不准，可以点击几次文字，
   或者按方向键移动光标。
```

### Q2：图片查看器在编辑器中能用吗？
```
A：图片查看器只在导出的HTML文件中可用，
   因为它是为最终文档设计的功能。
   在编辑器中，图片可以直接拖拽调整大小。
```

### Q3：链接插入后能修改吗？
```
A：可以的。选中链接，删除后重新插入即可。
   未来版本我们会添加链接编辑功能。
```
