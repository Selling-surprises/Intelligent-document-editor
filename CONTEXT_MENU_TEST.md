# 右键菜单功能测试清单

## 🔧 已修复的问题

### 1. 段落设置不生效
**问题**：选中文字 → 右键 → 段落 → 设置首行缩进 → 确定，没有任何变化

**原因**：
- 对话框打开时，原始选区丢失
- `applyParagraphSettings` 函数依赖 `window.getSelection()`，但选区已经不存在

**解决方案**：
- ✅ 添加 `currentElementRef` 保存当前操作的元素
- ✅ 添加 `savedSelectionRef` 保存当前选区
- ✅ 在打开对话框前保存元素和选区
- ✅ 在应用设置时使用保存的元素引用
- ✅ 添加详细的 console.log 用于调试

### 2. 表格设置不生效
**问题**：右键表格 → 表格属性 → 设置边框 → 确定，没有任何变化

**原因**：
- `applyTableSettings` 依赖 `menuState.context`，但对话框打开后上下文可能丢失

**解决方案**：
- ✅ 添加 `currentTableRef` 保存当前操作的表格
- ✅ 在打开对话框前保存表格引用
- ✅ 在应用设置时使用保存的表格引用
- ✅ 添加详细的 console.log 用于调试

## 🐛 调试技巧

### 1. 检查元素引用是否保存
打开浏览器控制台，右键点击文字选择"段落(P)..."，应该看到：
```
保存当前元素: <p>...</p>
```

### 2. 检查设置是否应用
在段落对话框中设置首行缩进为 2，点击确定，应该看到：
```
应用段落设置到元素: <p>...</p> 设置: {leftIndent: 0, rightIndent: 0, specialIndent: 'first-line', specialIndentValue: 2, ...}
```

### 3. 检查元素样式
在浏览器开发者工具中选中段落元素，查看 Styles 面板，应该看到：
```css
element.style {
  text-indent: 16px;
}
```

### 4. 检查表格引用
右键点击表格单元格选择"表格属性(R)..."，应该看到：
```
保存当前表格: <table>...</table>
```

### 5. 检查表格设置应用
在表格属性对话框中设置边框宽度为 2，点击确定，应该看到：
```
应用表格设置到表格: <table>...</table> 设置: {width: 100, widthUnit: '%', borderWidth: 2, ...}
```
