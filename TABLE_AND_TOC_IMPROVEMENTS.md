# 表格和目录功能改进说明 v3.11.0

## 📋 改进概述

本次更新主要改进了三个方面：
1. **表格列选择功能** - 自动高亮选中的列
2. **导出HTML目录交互** - 改为点击图标显示/隐藏
3. **表格工具栏位置** - 更贴近表格

---

## ✨ 1. 表格列选择功能

### 功能描述
当您点击表格中的任意单元格时，整列会自动高亮显示，让您清楚地知道当前操作的是哪一列。

### 视觉效果
```
┌─────────┬─────────┬─────────┬─────────┐
│  列1    │  列2    │  列3    │  列4    │
├─────────┼─────────┼─────────┼─────────┤
│  数据   │ [高亮]  │  数据   │  数据   │  ← 点击这个单元格
├─────────┼─────────┼─────────┼─────────┤
│  数据   │ [高亮]  │  数据   │  数据   │  ← 整列高亮
├─────────┼─────────┼─────────┼─────────┤
│  数据   │ [高亮]  │  数据   │  数据   │  ← 整列高亮
└─────────┴─────────┴─────────┴─────────┘
         ↑
      选中的列
```

### 使用方法

#### 选择列
1. 点击表格中的任意单元格
2. 整列会自动显示浅蓝色背景
3. 表格工具栏会出现在表格上方

#### 操作列
- **添加列**：点击"+ 列"按钮，在表格末尾添加新列
- **删除列**：点击"- 列"按钮，删除当前高亮的列
- **清空列**：点击"🗑️ 清空列"按钮，清空当前列所有单元格的内容
- **添加行**：点击"+ 行"按钮，在表格末尾添加新行
- **删除行**：点击"- 行"按钮，删除当前光标所在的行
- **删除表格**：点击"🗑️ 删除表格"按钮，删除整个表格

### 技术实现

#### 列高亮逻辑
```typescript
// 高亮指定列
const highlightColumn = (table: HTMLTableElement, columnIndex: number) => {
  // 先清除所有高亮
  clearAllHighlights(editor);
  
  // 高亮当前列的所有单元格
  for (let i = 0; i < table.rows.length; i++) {
    const cell = table.rows[i].cells[columnIndex];
    if (cell) {
      cell.style.backgroundColor = 'rgba(67, 97, 238, 0.1)';
      cell.setAttribute('data-column-selected', 'true');
    }
  }
};
```

#### 状态管理
- 使用`selectedColumn`状态存储当前选中的列索引
- 使用`data-column-selected`属性标记高亮的单元格
- 切换单元格时自动清除旧的高亮并应用新的高亮

## 🎯 2. 导出HTML目录交互改进

### 改进前
- 鼠标悬停在左侧边缘时目录自动滑出
- 鼠标离开后目录自动隐藏
- 交互不够明确，用户可能不知道有目录功能

### 改进后
- 左侧固定显示一个目录图标按钮（☰）
- 点击按钮显示/隐藏目录面板
- 交互更明确，用户一眼就能看到目录入口

### 视觉对比

#### 改进前
```
[隐藏的目录]  |  文档内容
              |
鼠标悬停 →    |
[目录滑出]    |  文档内容
```

#### 改进后
```
[☰]  |  文档内容
 ↑   |
图标 |

点击图标 →

[☰] [目录面板]  |  文档内容
     - 标题1    |
     - 标题2    |
     - 标题3    |
```

### 使用方法

#### 在导出的HTML中使用目录
1. 打开导出的HTML文件
2. 在左侧中间位置会看到一个蓝色的目录图标按钮（☰）
3. 点击按钮，目录面板会从左侧滑出
4. 点击目录中的标题，页面会平滑滚动到对应位置
5. 再次点击按钮，目录面板会隐藏

### 技术实现

#### CSS定位
```css
/* 目录按钮 - 固定在左侧中间 */
.toc-toggle {
  position: fixed;
  left: 0;
  top: 50%;
  transform: translateY(-50%);
  width: 40px;
  height: 40px;
  z-index: 1001;
}

/* 目录面板 - 默认隐藏在左侧 */
.floating-toc {
  position: fixed;
  left: 0;
  top: 50%;
  transform: translateY(-50%) translateX(-280px);
  width: 280px;
  transition: transform 0.3s ease;
  z-index: 1000;
}

/* 显示状态 */
.floating-toc.visible {
  transform: translateY(-50%) translateX(0);
}
```

#### JavaScript交互
```javascript
let tocVisible = false;

function toggleToc() {
  const toc = document.getElementById('floatingToc');
  if (!toc) return;
  tocVisible = !tocVisible;
  if (tocVisible) {
    toc.classList.add('visible');
  } else {
    toc.classList.remove('visible');
  }
}
```

## 🔧 3. 表格工具栏位置优化

### 改进内容
- 工具栏距离表格的距离从50px减少到45px
- 工具栏更贴近表格，操作更方便
- 减少视觉上的距离感

### 视觉对比

#### 改进前
```
                                    ← 50px 间距
[+ 行] [- 行] | [+ 列] [- 列] | [清空] [删除表格]
                                    ← 较大间距
┌─────────┬─────────┬─────────┐
│  列1    │  列2    │  列3    │
├─────────┼─────────┼─────────┤
│  数据   │  数据   │  数据   │
└─────────┴─────────┴─────────┘
```

#### 改进后
```
                                    ← 45px 间距
[+ 行] [- 行] | [+ 列] [- 列] | [清空列] [删除表格]
                                    ← 更紧凑
┌─────────┬─────────┬─────────┐
│  列1    │  列2    │  列3    │
├─────────┼─────────┼─────────┤
│  数据   │  数据   │  数据   │
└─────────┴─────────┴─────────┘
```

### 技术实现
```typescript
setPosition({
  top: rect.top - editorRect.top - 45, // 从 -50 改为 -45
  left: rect.left - editorRect.left,
});
```

## 🎨 视觉设计细节

### 列高亮颜色
- **颜色**：`rgba(67, 97, 238, 0.1)` - 浅蓝色，10%透明度
- **选择原因**：与主题色一致，不会过于突出，保持文字清晰可读

### 目录按钮设计
- **颜色**：`rgba(67, 97, 238, 0.95)` - 主题蓝色
- **图标**：☰ (汉堡菜单图标)
- **悬停效果**：放大1.1倍，颜色变为完全不透明
- **位置**：固定在左侧中间，不随页面滚动

### 工具栏设计
- **背景**：半透明白色卡片
- **阴影**：柔和的阴影效果
- **按钮**：Ghost样式，悬停时显示背景
- **分隔线**：使用细线分隔不同功能组

## 🔍 技术细节

### 列选择实现要点

1. **状态管理**
   - `selectedTable`: 当前选中的表格元素
   - `selectedColumn`: 当前选中的列索引
   - `visible`: 工具栏是否显示

2. **事件监听**
   - `selectionchange`: 监听选区变化
   - `click`: 监听点击事件
   - 清理函数：移除事件监听和清除高亮

3. **高亮管理**
   - 使用`data-column-selected`属性标记
   - 切换时先清除所有高亮
   - 再应用新的高亮

### 目录交互实现要点

1. **HTML结构**
   - 按钮和面板分离
   - 按钮使用fixed定位
   - 面板默认隐藏在左侧

2. **CSS动画**
   - 使用`transform`实现滑动效果
   - `transition`设置为0.3s ease
   - 使用`translateX`控制位置

3. **JavaScript逻辑**
   - 简单的布尔值切换
   - 添加/移除`visible`类
   - 平滑滚动到标题

## 💡 使用建议

### 表格编辑最佳实践
1. **规划表格结构**：先确定需要多少列，避免频繁添加/删除
2. **使用列高亮**：利用列高亮功能，确保操作正确的列
3. **批量操作**：使用"清空列"功能批量清空数据，而不是逐个单元格清空
4. **保持简洁**：表格列数不要太多，保持可读性

### 目录使用最佳实践
1. **合理使用标题层级**：H1用于主标题，H2用于章节，H3用于小节
2. **标题命名清晰**：使用描述性的标题，方便在目录中识别
3. **不要过度嵌套**：标题层级不要超过3级，保持目录简洁
4. **导出前检查**：导出前确保所有标题都正确设置

### 导出HTML最佳实践
1. **设置页面标题**：在设置中自定义页面标题
2. **上传Favicon**：添加网站图标，让导出的HTML更专业
3. **调整透明度**：根据背景图片调整内容区域透明度
4. **测试目录功能**：导出后测试目录是否正常工作
5. **检查表格样式**：确保表格在导出的HTML中显示正常

## 🐛 已知问题和解决方案

### 问题1：列高亮在某些情况下不显示
**原因**：表格样式可能覆盖了高亮样式
**解决方案**：确保表格单元格没有设置背景色，或者使用`!important`提高高亮样式优先级

### 问题2：目录按钮在某些浏览器中位置不对
**原因**：不同浏览器对`fixed`定位的支持可能有差异
**解决方案**：使用标准的CSS定位，避免使用浏览器特定的属性

### 问题3：工具栏位置在滚动时不准确
**原因**：工具栏位置计算基于表格的`getBoundingClientRect()`
**解决方案**：在滚动时重新计算工具栏位置，或者使用`position: sticky`
