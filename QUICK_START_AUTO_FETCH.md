# 链接卡片自动获取 - 快速上手

## 🚀 5秒钟学会使用

### 步骤1：打开插入链接对话框
点击工具栏的 **🔗 插入链接** 按钮

### 步骤2：选择卡片链接
选择 **卡片链接（精美卡片样式）**

### 步骤3：输入链接地址
在"链接地址"输入框中输入网址，例如：
```
https://www.example.com
```

### 步骤4：点击自动获取
点击右侧的 **✨ 自动获取** 按钮

### 步骤5：等待填充
系统会自动填充：
- 📝 卡片标题
- 📄 卡片描述
- 🖼️ 卡片图片

### 步骤6：插入链接
点击 **插入链接** 按钮完成

---

## 💡 提示

### ✅ 成功获取后
- 可以手动修改自动填充的内容
- 所有字段都可以编辑
- 图片URL可以更换

### ❌ 获取失败时
- 手动填写标题和描述
- 图片URL可以留空
- 不影响卡片插入

---

## 🎯 示例

### 示例1：插入新闻链接
```
链接：https://www.bbc.com/news/article
↓ 点击自动获取
标题：BBC News - Latest Headlines
描述：Breaking news from around the world
图片：https://www.bbc.com/news/image.jpg
```

### 示例2：插入博客链接
```
链接：https://medium.com/@author/article
↓ 点击自动获取
标题：How to Build a Great Product
描述：A comprehensive guide to product development
图片：https://miro.medium.com/image.jpg
```

---

## ⚡ 快捷技巧

1. **批量插入**：对多个链接重复上述步骤
2. **快速修改**：获取后立即修改不准确的内容
3. **手动备份**：获取失败时手动填写
