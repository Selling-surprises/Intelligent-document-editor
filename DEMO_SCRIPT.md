## 演示1：JSON导入功能（使用示例文档）

### 步骤

1. **打开设置面板**
   - 点击右上角的"设置"按钮（⚙️图标）
   - 设置面板从右侧滑出

2. **导入示例文档**
   - 滚动到设置面板底部
   - 找到"导入JSON"和"导出JSON"按钮
   - 点击"导入JSON"按钮
   - 在文件选择对话框中选择 `example.json`
   - 点击"打开"

3. **验证导入结果**
   - 编辑器中应该显示示例内容
   - 包含标题、段落、列表、表格和链接
   - 页面标题应该变为"示例文档"
   - 应该显示"导入成功"的提示消息

### 预期结果

✅ 编辑器显示完整的示例内容  
✅ 页面标题更新为"示例文档"  
✅ 显示成功提示消息  
✅ 所有格式正确保留

## 演示2：链接插入功能（选中文字）

### 步骤

1. **输入文字**
   - 在编辑器中输入："访问我们的网站了解更多信息"

2. **选中文字**
   - 用鼠标选中"我们的网站"这几个字
   - 文字应该高亮显示

3. **插入链接**
   - 点击工具栏的链接按钮（🔗图标）
   - 链接对话框打开
   - 在"链接地址"输入框中输入：`https://example.com`
   - 注意：不需要输入"显示文本"，因为已经选中了文字
   - 点击"插入链接"按钮

4. **验证结果**
   - "我们的网站"应该变成蓝色链接
   - 链接文本应该是"我们的网站"
   - 应该显示"链接插入成功"的提示消息

### 预期结果

✅ 选中的文字变成链接  
✅ 链接文本是选中的文字  
✅ 链接地址正确  
✅ 显示成功提示消息

3. **输入链接信息**

   - 链接应该指向 https://github.com
   - 应该显示"链接插入成功"的提示消息

### 预期结果

✅ 链接文本是输入的文本

✅ 显示成功提示消息

## 演示4：JSON导出功能

### 步骤

1. **创建一些内容**
   - 在编辑器中输入一些文字
   - 插入一个表格
   - 插入一个链接
   - 设置页面标题为"我的文档"

2. **导出JSON**
   - 点击右上角的"设置"按钮
   - 滚动到设置面板底部
   - 点击"导出JSON"按钮
   - JSON文件应该自动下载

3. **验证导出文件**
   - 检查下载的文件名应该是"我的文档.json"
   - 用文本编辑器打开JSON文件
   - 验证文件包含：
     - `version`: "5.1.0"
     - `content`: HTML内容
     - `settings`: 页面设置
     - `exportDate`: 导出时间

### 预期结果

✅ JSON文件成功下载  
✅ 文件名使用页面标题  
✅ JSON格式正确  
✅ 包含所有必要字段  
✅ 显示成功提示消息

## 演示5：表格边框样式

### 步骤

1. **插入表格**
   - 点击工具栏的表格按钮
   - 选择3行3列
   - 点击"插入表格"
   - 表格应该有明显的2px实线边框

2. **修改边框粗细**
   - 右键点击表格中的任意单元格
   - 选择"边框样式" → "边框粗细" → "粗线（3px）"
   - 表格边框应该变粗

3. **修改边框样式**
   - 右键点击表格中的任意单元格
   - 选择"边框样式" → "边框样式" → "虚线"
   - 表格边框应该变成虚线

### 预期结果

✅ 表格插入成功  
✅ 默认边框清晰可见  
✅ 边框粗细可以修改  
✅ 边框样式可以修改  
✅ 显示成功提示消息

3. **清空内容**

5. **导出HTML**
   - 点击"设置" → "导出为HTML文件"
   - 保存HTML文件
   - 用浏览器打开HTML文件
   - 验证所有内容正确显示

## 常见问题演示

### Q1: JSON文件和HTML文件有什么区别？

**演示：**
1. 导出JSON文件，用文本编辑器打开
2. 导出HTML文件，用浏览器打开
3. 说明：
   - JSON用于保存和恢复编辑状态
   - HTML用于查看和分享

### Q2: 如何在不同设备间同步文档？

**演示：**
1. 在设备A上导出JSON
2. 通过邮件或云盘传输JSON文件
3. 在设备B上导入JSON
4. 所有内容和设置完整恢复

### Q3: 链接插入后为什么不能点击？

**演示：**
1. 在编辑器中插入链接
2. 说明：编辑模式下链接不可点击
3. 导出为HTML
4. 在浏览器中打开HTML
5. 链接可以正常点击
