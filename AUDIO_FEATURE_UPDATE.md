# 音频和视频功能更新说明

## 更新内容

### 1. 新增可选元数据字段

在音频插入对话框中添加了以下可选字段：
- **歌曲名称**：可自定义显示的歌曲标题
- **歌手/艺术家**：可自定义显示的艺术家名称
- **封面图片链接**：可自定义显示的封面图片URL

这些字段都是**可选的**，不是强制要求。如果不填写，系统会使用默认值或不显示。

### 2. 新增删除按钮（音频和视频）

为所有插入的音频和视频添加了删除按钮：
- 删除按钮位于音频/视频播放器的右上角
- 按钮样式：红色背景，白色文字，带有悬停效果
- 点击删除按钮可以直接移除音频/视频元素
- 支持所有视频平台：YouTube、Bilibili、优酷、腾讯视频、爱奇艺、直接链接
- 支持所有音频平台：网易云音乐、QQ音乐、Spotify、直接链接

### 3. 优化对话框滚动体验

修复了音频插入对话框的滚动问题：
- 对话框最大高度限制为视口的90%
- 内容区域支持垂直滚动
- 底部按钮固定显示，不会被遮挡
- 提供流畅的滚动条体验

### 4. 自定义音频播放器

当用户填写了歌名、歌手或封面信息时，系统会生成一个美观的自定义音频播放器：
- 渐变紫色背景
- 显示封面图片（如果提供）
- 显示歌曲名称和艺术家
- 内置HTML5音频控制器
- 带有删除按钮

### 5. 平台嵌入式播放器

对于网易云音乐、QQ音乐、Spotify等平台：
- 保持原有的iframe嵌入式播放器
- 在播放器右上角添加删除按钮
- 删除按钮不影响播放器的正常使用

## 使用方法

### 插入基本音频
1. 点击工具栏的音乐图标
2. 选择音频平台（直接链接、网易云音乐、QQ音乐、Spotify）
3. 输入音频链接
4. 点击"插入"按钮

### 插入带元数据的音频
1. 点击工具栏的音乐图标
2. 选择"直接链接"平台
3. 输入音频链接（必填）
4. 在"可选信息"区域填写：
   - 歌曲名称（例如：夜曲）
   - 歌手/艺术家（例如：周杰伦）
   - 封面图片链接（例如：https://example.com/cover.jpg）
5. 点击"插入"按钮

### 删除音频
- 将鼠标悬停在音频播放器上
- 点击右上角的红色"删除"按钮
- 音频元素将被立即移除

### 删除视频
- 将鼠标悬停在视频播放器上
- 点击右上角的红色"删除"按钮
- 视频元素将被立即移除

## 注意事项

1. 封面图片链接必须是有效的URL格式
2. 自定义音频播放器仅在提供了至少一个元数据字段时显示
3. 删除按钮使用内联JavaScript，确保在导出HTML时功能正常
4. 所有音频和视频容器都设置了 `contenteditable="false"`，防止意外编辑
5. 每个音频和视频元素都有唯一的ID，便于管理和操作

## 兼容性

- 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
- 自定义音频播放器使用HTML5 audio标签
- 视频播放器支持HTML5 video标签和各平台iframe嵌入
- 删除功能使用原生JavaScript，无需额外依赖
