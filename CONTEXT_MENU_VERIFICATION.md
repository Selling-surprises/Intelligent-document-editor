# 右键菜单功能验证指南

## 🎯 快速测试步骤

### 测试 1：段落首行缩进

1. **在编辑器中输入文字**
   ```
   这是一段测试文字。这是一段测试文字。这是一段测试文字。这是一段测试文字。
   ```

2. **选中文字**
   - 用鼠标拖动选中整段文字
   - 或者三击选中整段

3. **打开右键菜单**
   - 在选中的文字上右键点击
   - 应该看到右键菜单弹出

4. **打开段落对话框**
   - 点击菜单中的"段落(P)..."
   - 应该看到段落设置对话框弹出
   - **打开浏览器控制台（F12）**，应该看到类似这样的日志：
     ```
     执行命令: paragraphDialog ...
     ```

5. **设置首行缩进**
   - 在对话框中找到"特殊格式"下拉框
   - 选择"首行缩进"
   - 在"度量值"输入框中输入 `2`
   - 点击"确定"按钮

6. **验证效果**
   - 段落的第一行应该向右缩进 2 个字符（16px）
   - **在浏览器控制台中**，应该看到类似这样的日志：
     ```
     应用段落设置到元素: <p>...</p> 设置: {leftIndent: 0, rightIndent: 0, specialIndent: 'first-line', specialIndentValue: 2, ...}
     ```
   - **在开发者工具的 Elements 面板中**，选中这个段落元素，查看 Styles 面板，应该看到：
     ```css
     element.style {
       text-indent: 16px;
     }
     ```

---

### 测试 2：表格边框设置

1. **插入表格**
   - 使用工具栏的表格按钮插入一个 3x3 的表格

2. **打开右键菜单**
   - 在表格的任意单元格上右键点击
   - 应该看到表格专用的右键菜单

3. **打开表格属性对话框**
   - 点击菜单中的"表格属性(R)..."
   - 应该看到表格属性对话框弹出
   - **打开浏览器控制台（F12）**，应该看到类似这样的日志：
     ```
     执行命令: tableProperties ...
     ```

4. **设置边框**
   - 切换到"边框"标签页
   - 边框宽度：输入 `3`
   - 边框样式：选择"实线"
   - 边框颜色：选择红色（#ff0000）
   - 点击"确定"按钮

5. **验证效果**
   - 表格和所有单元格的边框应该变为 3px 红色实线
   - **在浏览器控制台中**，应该看到类似这样的日志：
     ```
     应用表格设置到表格: <table>...</table> 设置: {width: 100, widthUnit: '%', borderWidth: 3, borderStyle: 'solid', borderColor: '#ff0000', ...}
     ```
   - **在开发者工具的 Elements 面板中**，选中表格元素，查看 Styles 面板，应该看到：
     ```css
     element.style {
       border: 3px solid rgb(255, 0, 0);
     }
     ```

---

### 测试 3：插入表格行

1. **在表格中右键点击**
   - 在表格的任意单元格上右键点击

2. **插入行**
   - 点击"插入(I)" → "在上方插入行"
   - 应该在当前行上方插入一行新行

3. **验证效果**
   - 表格应该多了一行
   - 新行应该在当前行的上方

---

### 测试 4：合并单元格

1. **选中多个单元格**
   - 点击第一个单元格
   - 按住 Shift 键，点击第二个单元格
   - 两个单元格应该都被选中（有蓝色高亮）

2. **打开右键菜单**
   - 在选中的单元格上右键点击

3. **合并单元格**
   - 点击"合并单元格(M)"
   - 两个单元格应该合并为一个

4. **验证效果**
   - 原来的两个单元格变成了一个
   - 内容合并在一起

## 🐛 如果功能不生效

### 问题 1：段落设置不生效

**症状**：
- 段落对话框能打开
- 设置了首行缩进
- 点击确定后没有任何变化

**调试步骤**：

1. **打开浏览器控制台（F12）**

2. **查看日志**
   - 右键点击文字 → 段落(P)...
   - 应该看到：`执行命令: paragraphDialog ...`
   - 如果没有看到这个日志，说明右键菜单没有正确触发

3. **设置首行缩进并点击确定**
   - 应该看到：`应用段落设置到元素: <p>...</p> 设置: {...}`
   - 如果没有看到这个日志，说明 `applyParagraphSettings` 函数没有被调用

4. **检查元素引用**
   - 如果看到：`未找到要应用段落设置的元素`
   - 说明元素引用丢失了
   - 这是一个 bug，需要修复

5. **检查元素样式**
   - 在 Elements 面板中选中段落元素
   - 查看 Styles 面板
   - 如果看到 `text-indent: 16px`，说明样式已经应用
   - 如果没有看到，说明样式应用失败

**可能的原因**：
- 元素引用在对话框打开时丢失
- 选区在对话框打开时丢失
- 元素不是 `<p>` 或 `<div>` 标签

**解决方案**：
- 已经在代码中添加了元素引用保存机制
- 如果仍然不生效，请提供控制台日志截图

### 问题 2：表格设置不生效

**症状**：
- 表格属性对话框能打开
- 设置了边框
- 点击确定后没有任何变化

**调试步骤**：

1. **打开浏览器控制台（F12）**

2. **查看日志**
   - 右键点击表格 → 表格属性(R)...
   - 应该看到：`执行命令: tableProperties ...`

3. **设置边框并点击确定**
   - 应该看到：`应用表格设置到表格: <table>...</table> 设置: {...}`
   - 如果没有看到这个日志，说明 `applyTableSettings` 函数没有被调用

4. **检查表格引用**
   - 如果看到：`未找到要应用设置的表格`
   - 说明表格引用丢失了

5. **检查表格样式**
   - 在 Elements 面板中选中表格元素
   - 查看 Styles 面板
   - 如果看到 `border: 3px solid ...`，说明样式已经应用

---

### 问题 3：插入行列不生效

**症状**：
- 右键菜单能显示
- 点击"插入行"没有反应

**调试步骤**：

1. **打开浏览器控制台（F12）**

2. **查看日志**
   - 右键点击表格 → 插入(I) → 在上方插入行
   - 应该看到：`执行命令: insertRow ...`

3. **检查上下文信息**
   - 日志中应该包含 `context` 信息
   - 检查 `context.table` 和 `context.row` 是否存在
   - 如果不存在，说明上下文检测有问题

---

### 问题 4：合并单元格不生效

**症状**：
- 选中了多个单元格
- 点击"合并单元格"没有反应

**调试步骤**：

1. **检查单元格是否真的被选中**
   - 选中的单元格应该有蓝色高亮
   - 如果没有高亮，说明选中失败

2. **查看控制台日志**
   - 应该看到：`执行命令: mergeCells ...`
   - 检查 `context.cells` 是否包含多个单元格

3. **检查合并条件**
   - 只能合并同一行或同一列的单元格
   - 不能合并已经合并过的单元格

## 🔍 高级调试

### 手动测试元素引用

在浏览器控制台中运行：

```javascript
// 1. 获取编辑器元素
const editor = document.querySelector('.editor-content');
console.log('编辑器元素:', editor);

// 2. 获取第一个段落
const paragraph = editor.querySelector('p');
console.log('段落元素:', paragraph);

// 3. 手动应用样式
paragraph.style.textIndent = '32px';
console.log('手动设置首行缩进为 32px');

// 4. 检查样式
console.log('段落样式:', paragraph.style.textIndent);
```

### 手动测试表格引用

```javascript
// 1. 获取表格
const table = document.querySelector('table');
console.log('表格元素:', table);

// 2. 手动应用样式
table.style.border = '5px solid blue';
console.log('手动设置边框为 5px 蓝色实线');

// 3. 检查样式
console.log('表格样式:', table.style.border);
```
