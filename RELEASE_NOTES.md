# 发布说明 - v5.1.0

## 🎉 新版本发布

离线富文本编辑器 v5.1.0 现已发布！本次更新带来了JSON导入导出功能、链接插入功能修复以及表格边框样式增强。

## ✨ 主要更新

### 1. JSON导入导出功能

现在您可以将文档内容和所有设置保存为JSON文件，方便下次继续编辑或在不同设备间迁移文档。

**如何使用：**
- 点击右上角"设置"按钮
- 点击"导出JSON"保存当前文档
- 点击"导入JSON"恢复之前保存的文档

**JSON文件包含：**
- 文档内容（文字、表格、链接等）
- 页面设置（标题、图标、背景图片、透明度）
- 版本信息和导出时间

### 2. 链接插入功能修复

修复了链接插入功能的多个问题，现在可以更方便地插入链接。

**改进：**
- ✅ 支持选中文字后插入链接
- ✅ 自动使用选中的文字作为链接文本
- ✅ 修复了选区丢失的问题
- ✅ 链接插入位置更准确

**使用方法：**
1. 选中要添加链接的文字
2. 点击工具栏的链接按钮
3. 输入链接地址
4. 点击"插入链接"

### 3. 表格边框样式增强

表格边框样式功能已在之前版本实现，本次更新优化了默认样式。

**功能：**
- 支持3种边框粗细（1px、2px、3px）
- 支持4种边框样式（实线、虚线、点线、双线）
- 默认使用2px实线边框，更加清晰

## 🚀 快速开始

### 新用户

1. 打开编辑器
2. 点击"设置" → "导入JSON"
3. 选择 `example.json` 文件
4. 查看示例内容，了解编辑器功能

### 现有用户

1. 导出当前文档为JSON（备份）
2. 刷新页面体验新功能
3. 导入JSON恢复文档

## 💡 使用技巧

### JSON导入导出

**最佳实践：**
- 定期导出JSON文件作为备份
- 使用有意义的页面标题（会作为文件名）
- 在不同设备间使用JSON文件同步文档

**使用场景：**
- 保存编辑进度，下次继续编辑
- 在不同设备间迁移文档
- 备份重要文档
- 分享文档给他人编辑

### 链接插入

**推荐方式：**
1. 先输入或粘贴文字
2. 选中要添加链接的文字
3. 点击链接按钮
4. 输入链接地址

**优势：**
- 更直观
- 更快捷
- 不容易出错

### 表格边框

**建议：**
- 数据表格：使用细线（1px）或中等（2px）
- 强调表格：使用粗线（3px）或双线
- 草稿表格：使用虚线或点线
