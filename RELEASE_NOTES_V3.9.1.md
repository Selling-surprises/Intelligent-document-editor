# 离线Word文档编辑器 v3.9.1 - 修复版本发布说明

## 🎯 版本概述

v3.9.1是一个重要的修复版本，解决了v3.9.0中用户反馈的三个关键问题，并新增了插入目录功能。

## 🐛 问题修复

### 1. 修复文件选择按钮样式问题

**问题描述**：
- 在设置面板中，文件选择按钮的颜色与背景融为一体
- 用户无法清楚地看到按钮轮廓
- 影响用户体验

**解决方案**：
- 为所有文件输入框添加了明显的样式
- 文件选择按钮现在使用主题色（蓝色）背景
- 按钮文字使用白色，对比度高
- 添加了hover效果，鼠标悬停时颜色变深

**效果**：
```
✅ Favicon上传按钮：蓝色背景，白色文字
✅ 背景图片上传按钮：蓝色背景，白色文字
✅ 鼠标悬停时：颜色加深，提供视觉反馈
```

### 2. 修复表格插入失焦问题

**问题描述**：
- 点击表格插入按钮后，对话框打开
- 编辑器失去焦点，光标位置丢失
- 点击"插入表格"后，表格无法插入到正确位置
- 有时表格会插入到文档末尾或开头

**解决方案**：
- 实现了光标位置保存机制
- 在编辑器中添加了`onMouseUp`和`onKeyUp`事件监听
- 每次用户操作后自动保存光标位置
- 插入表格时先恢复光标位置，再执行插入

**技术实现**：
```typescript
// 保存光标位置
const saveSelection = () => {
  const selection = window.getSelection();
  if (selection && selection.rangeCount > 0) {
    savedRangeRef.current = selection.getRangeAt(0).cloneRange();
  }
};

// 恢复光标位置
const restoreSelection = () => {
  if (editor && savedRangeRef.current) {
    editor.focus();
    const selection = window.getSelection();
    selection?.removeAllRanges();
    selection?.addRange(savedRangeRef.current);
  }
};
```

**效果**：
```
✅ 打开对话框前：自动保存光标位置
✅ 插入表格时：恢复到正确的光标位置
✅ 表格插入：准确插入到用户期望的位置
```

### 3. 新增插入目录功能

**问题描述**：
- 用户反馈没有插入目录的按钮
- 虽然左侧有自动生成的目录面板，但无法将目录插入到文档中
- 导出的HTML文件中没有目录

**解决方案**：
- 在工具栏添加了"插入目录"按钮（📑图标）
- 点击按钮可将当前文档的目录结构插入到光标位置
- 目录会自动识别所有H1-H3标题
- 生成带层级缩进和链接的目录

**功能特点**：
- ✅ 自动识别标题：扫描文档中的所有H1-H3标题
- ✅ 层级缩进：根据标题级别自动缩进
- ✅ 可点击链接：目录中的每个项目都是可点击的链接
- ✅ 美观样式：带边框、圆角、背景色的专业样式
- ✅ 智能提示：如果没有标题，会提示用户先添加标题

**使用方法**：
1. 在文档中添加一些标题（H1-H3）
2. 将光标放在想要插入目录的位置
3. 点击工具栏中的目录图标（📑）
4. 目录会自动插入到光标位置

**目录样式**：
```
┌─────────────────────────────┐
│ 📑 目录                      │
├─────────────────────────────┤
│ • 第一章标题 (H1)            │
│   •• 第一节标题 (H2)         │
│     ••• 小节标题 (H3)        │
│ • 第二章标题 (H1)            │
│   •• 第二节标题 (H2)         │
└─────────────────────────────┘
```

**效果**：
```
✅ 目录自动生成：基于文档中的标题
✅ 层级清晰：使用缩进和符号区分层级
✅ 可点击跳转：点击目录项可跳转到对应标题
✅ 导出保留：导出HTML时目录会保留在文档中
```

## 📚 使用指南

### 插入目录的完整流程

**步骤1：添加标题**
```
1. 输入文字
2. 选中文字
3. 在工具栏选择"标题 1"、"标题 2"或"标题 3"
4. 重复以上步骤添加多个标题
```

**步骤2：插入目录**
```
1. 将光标放在想要插入目录的位置（通常是文档开头）
2. 点击工具栏中的目录图标（📑）
3. 目录会自动插入
4. 可以点击目录中的链接跳转到对应标题
```

**步骤3：导出文档**
```
1. 点击设置面板中的"导出为HTML文件"
2. 导出的文件会包含目录
3. 在浏览器中打开，目录链接可正常工作
```

## ⚠️ 注意事项

### 插入目录
- 必须先添加标题（H1-H3）才能插入目录
- 如果没有标题，会显示提示信息
- 目录会插入到当前光标位置
- 可以多次插入目录（例如在不同章节）

### 文件上传
- 文件选择按钮现在更加醒目
- 支持本地上传和在线URL两种方式
- 使用Tab键可以快速切换

### 表格插入
- 确保在编辑区域点击后再打开表格对话框
- 表格会插入到光标位置
- 可以直接在表格单元格中编辑内容
