# 版本 3.12.2 更新总结

## 📅 发布日期
2025-12-06

## 🎯 版本概述

版本3.12.2是一个功能增强版本，主要新增了增加/减少缩进功能，支持工具栏按钮和键盘快捷键两种操作方式，提升文档编辑的灵活性和效率。

---

## ✨ 新增功能

### 1. 增加缩进功能

#### 功能描述
支持通过工具栏按钮或Tab键增加选中文本或段落的缩进。

#### 核心特性
- ✅ **工具栏按钮**：直观的向右箭头图标
- ✅ **键盘快捷键**：Tab键快速操作
- ✅ **多行支持**：可同时缩进多行文本
- ✅ **列表支持**：创建多层级列表结构
- ✅ **实时生效**：操作立即显示效果

#### 使用场景
1. **段落缩进**：调整段落的左边距
2. **列表层级**：创建子列表和多层级结构
3. **代码格式化**：调整代码块的缩进
4. **引用文本**：创建视觉上的引用效果

---

### 2. 减少缩进功能

#### 功能描述
支持通过工具栏按钮或Shift+Tab键减少选中文本或段落的缩进。

#### 核心特性
- ✅ **工具栏按钮**：直观的向左箭头图标
- ✅ **键盘快捷键**：Shift+Tab键快速操作
- ✅ **多行支持**：可同时减少多行缩进
- ✅ **列表支持**：调整列表层级
- ✅ **实时生效**：操作立即显示效果

#### 使用场景
1. **恢复原位**：取消段落缩进
2. **调整层级**：将子列表提升为同级
3. **代码格式化**：减少代码块的缩进
4. **修正错误**：撤销误操作的缩进

### 2. 键盘快捷键

**文件**：`src/components/editor/EditorContent.tsx`

**新增内容**：
```tsx
const handleKeyDown = (e: React.KeyboardEvent) => {
  // Tab键：增加缩进
  if (e.key === 'Tab' && !e.shiftKey) {
    e.preventDefault();
    document.execCommand('indent', false);
  }
  // Shift+Tab：减少缩进
  else if (e.key === 'Tab' && e.shiftKey) {
    e.preventDefault();
    document.execCommand('outdent', false);
  }
};
```

**实现细节**：
- 监听contentEditable div的onKeyDown事件
- 阻止Tab键的默认行为（切换焦点）
- 使用document.execCommand执行缩进命令

---

### 3. execCommand API

**使用的命令**：
- `indent`：增加缩进，通常添加40px的左边距
- `outdent`：减少缩进，移除左边距

**浏览器支持**：
- ✅ Chrome 120+
- ✅ Firefox 121+
- ✅ Safari 17+
- ✅ Edge 120+

**注意事项**：
- execCommand API已被标记为过时，但仍被广泛支持
- 未来可能需要迁移到新的API
