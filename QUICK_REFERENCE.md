# 快速参考指南

## 📱 移动端适配功能

### 快速开始

```
1. 打开设置面板
2. 找到"移动端设置"分组
3. 确认"启用移动端适配"开关已打开
4. （可选）上传移动端背景图片
5. 导出HTML文件
```

### 开关状态

| 状态 | 效果 | 移动端背景 |
|------|------|-----------|
| ✅ 开启 | 移动端优化样式 | 可上传 |
| ❌ 关闭 | 使用桌面端样式 | 不可用 |

---

## 🎴 链接卡片功能

### 插入链接

```
1. 选中文字（可选）
2. 点击工具栏"插入链接"按钮
3. 选择链接类型：
   - 文本链接：传统超链接
   - 卡片链接：现代卡片样式
4. 填写必填信息
5. 点击"插入"
```

### 卡片字段

| 字段 | 必填 | 说明 |
|------|------|------|
| 链接地址 | ✅ | 完整的URL |
| 卡片标题 | ✅ | 显示的标题 |
| 卡片描述 | ❌ | 简短描述 |
| 卡片图片 | ❌ | 图片URL |

## 🐛 常见问题

### Q1：移动端适配开关在哪里？

**A**：在设置面板的"移动端设置"分组中，开关始终可见。

### Q2：如何测试移动端效果？

**A**：
1. 导出HTML文件
2. 在浏览器中打开
3. 按F12打开开发者工具
4. 按Ctrl+Shift+M切换到移动设备模式

### Q3：移动端背景图片不显示？

**A**：检查以下几点：
- 是否启用了移动端适配
- 是否上传了移动端背景图片
- 图片URL是否正确
- 是否在移动设备模式下查看（屏幕宽度≤768px）

### Q4：如何移除背景图片？

**A**：在图片预览区域点击"移除"按钮。

### Q5：导出的HTML文件很大？

**A**：
- 压缩背景图片
- 使用WebP格式
- 减少图片数量
- 优化图片尺寸

## 💡 使用技巧

### 技巧1：快速配置

```
1. 配置好一套设置
2. 导出JSON文件
3. 下次使用时导入JSON
4. 立即恢复所有设置
```

### 技巧2：多设备适配

```
1. 上传横向桌面端背景
2. 上传竖向移动端背景
3. 启用移动端适配
4. 导出HTML
5. 在任何设备上完美显示
```

### 技巧3：图片优化

```
1. 使用在线工具压缩图片
   - TinyPNG
   - Squoosh
   - ImageOptim
2. 转换为WebP格式
3. 调整到合适尺寸
4. 上传到编辑器
```

### 技巧4：链接卡片

```
1. 重要链接使用卡片模式
2. 添加描述和图片
3. 提高点击率
4. 增强视觉效果
```

### 技巧5：内容组织

```
1. 使用H1-H3标题
2. 自动生成目录
3. 方便导航
4. 提升可读性
```
