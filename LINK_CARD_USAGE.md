# 链接卡片使用指南

## 📖 快速上手

### 插入链接卡片

1. 点击工具栏的"🔗 插入链接"按钮
2. 在弹出的对话框中：
   - 输入链接地址（必填）
   - 选择"卡片链接"类型
   - 输入标题（可选，默认使用链接地址）
   - 输入描述（可选）
   - 输入图片URL（可选）
3. 点击"插入"按钮

### 编辑链接卡片

1. 将鼠标悬停在链接卡片上
2. 右上角会显示"编辑"和"删除"按钮
3. 点击"编辑"按钮
4. 依次输入新的：
   - 链接地址
   - 标题
   - 描述（可选）
5. 确认后卡片内容会立即更新

### 删除链接卡片

1. 将鼠标悬停在链接卡片上
2. 点击右上角的"删除"按钮
3. 在确认对话框中点击"确定"
4. 卡片会被删除

### 点击链接卡片

- 点击卡片的内容区域（图片或文字）会在新标签页打开链接
- 点击编辑或删除按钮不会触发链接跳转

## 💡 使用技巧

### 技巧1：快速修正错误

如果插入卡片后发现链接地址错误，不需要删除重新插入：
1. 悬停在卡片上
2. 点击"编辑"
3. 只修改链接地址
4. 其他内容保持不变

### 技巧2：更新卡片信息

如果链接的标题或描述需要更新：
1. 悬停在卡片上
2. 点击"编辑"
3. 修改标题和描述
4. 链接地址保持不变

### 技巧3：导出后编辑

导出的HTML文件也支持编辑和删除：
1. 在浏览器中打开导出的HTML
2. 悬停在卡片上显示按钮
3. 编辑或删除卡片
4. 使用浏览器的"保存页面"功能保存修改

### 技巧4：批量管理

如果需要删除多个卡片：
1. 逐个悬停并点击删除
2. 或者使用撤销功能恢复误删的卡片

## ⚠️ 注意事项

### 编辑器内

- ✅ 编辑和删除会触发内容更新
- ✅ 可以使用撤销功能恢复
- ✅ 会显示Toast提示

### 导出HTML

- ⚠️ 编辑后需要重新保存HTML文件
- ⚠️ 浏览器不会自动保存修改
- ⚠️ 删除后无法撤销（除非刷新页面）

### 最佳实践

- 📝 建议在编辑器中完成所有编辑
- 💾 编辑完成后再导出HTML
- 🔍 导出前检查所有卡片内容
- 📤 导出HTML主要用于分享和展示

## 🎯 常见问题

### Q: 为什么悬停时看不到按钮？

A: 可能的原因：
1. 卡片不是通过新版本创建的（旧版本的卡片没有按钮）
2. 浏览器缓存问题（刷新页面试试）
3. CSS样式被覆盖（检查是否有自定义样式）

### Q: 编辑后内容没有更新？

A: 可能的原因：
1. 点击了取消按钮
2. 输入的内容为空
3. 浏览器阻止了弹出对话框

### Q: 删除后能恢复吗？

A: 
- 编辑器内：可以使用撤销功能（Ctrl+Z）恢复
- 导出HTML：无法恢复，除非刷新页面

### Q: 导出的HTML编辑后如何保存？

A: 
1. 在浏览器中右键点击页面
2. 选择"另存为"或"保存页面"
3. 选择保存位置和文件名
4. 保存为HTML文件
