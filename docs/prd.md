# 基于 Canvas 的 Word 文档编辑器右键菜单与对话框需求文档

## 1. 工具概述

### 1.1 工具名称
基于 Canvas 的 Word 文档编辑器右键菜单与对话框系统

### 1.2 工具描述
为现有的 Word 文档编辑器添加上下文敏感的右键菜单系统,支持段落、表格、图片等不同元素的右键操作,提供与 Microsoft Word 相似的用户体验。包含段落对话框、表格属性对话框等交互组件。

## 2. 核心功能

### 2.1 右键菜单系统

#### 2.1.1 上下文检测
- 根据点击位置自动识别上下文类型:段落文本、表格单元格、图片、空白区域
- 支持表格内不同区域的识别:单元格内部、表格边框线、单元格交界处
- 检测选区状态,调整菜单显示内容

#### 2.1.2 菜单结构
- 主菜单:根据上下文动态生成菜单项
- 子菜单:支持多级菜单展开
- 分隔线:清晰分组不同功能区域
- 禁用状态:灰色显示不可用的菜单项
- 快捷键提示:在菜单项后显示对应的快捷键(如 Ctrl+C)

#### 2.1.3 菜单项交互
- 点击执行对应命令
- 悬停显示高亮效果
- 支持子菜单展开
- 点击其他区域自动关闭菜单

### 2.2 段落右键菜单

#### 2.2.1 核心功能
- 剪切、复制、粘贴
- 段落对话框(打开段落设置窗口)
- 项目符号:多种项目符号样式
- 编号:多种编号样式
- 多级列表
- 字体对话框(打开字体设置窗口)
- 插入:分页符、分栏符等
- 删除
- 全选
- 新建批注
- 超链接
- 样式:应用预定义样式

### 2.3 表格右键菜单

#### 2.3.1 核心功能
- 剪切、复制、粘贴单元格
- 插入:插入行/列(支持在上方/下方/左侧/右侧插入)
- 删除单元格:删除单元格/删除行/删除列/删除表格
- 选择:选择单元格/行/列/表格
- 合并单元格
- 拆分单元格(打开拆分对话框)
- 表格属性对话框(打开表格属性设置窗口)
- 边框和底纹
- 单元格对齐方式:9 种对齐方式(上/中/下 左/中/右)
- 自动调整:根据内容调整表格/根据窗口调整表格/固定列宽
- 文字方向
- 排序
- 公式

### 2.4 图片右键菜单

#### 2.4.1 核心功能
- 剪切、复制、粘贴
- 更改图片
- 另存为图片
- 大小和位置对话框(精确设置图片尺寸和位置)
- 设置图片格式
- 环绕文字:7 种环绕方式(嵌入型/四周型/紧密型/穿越型/上下型环绕/衬于文字下方/浮于文字上方)
- 对齐方式
- 旋转:向右旋转 90°/向左旋转 90°
- 翻转:垂直翻转/水平翻转
- 裁剪
- 插入题注
- 超链接

### 2.5 空白区域右键菜单

#### 2.5.1 核心功能
- 粘贴
- 段落对话框
- 制表位对话框
- 插入符号
- 全选
- 字体对话框

### 2.6 对话框系统

#### 2.6.1 段落对话框

**功能要求:**
- 缩进设置:
  - 左侧缩进:输入数字 + 单位选择(字符/厘米/磅)
  - 右侧缩进:输入数字 + 单位选择(字符/厘米/磅)
  - 特殊格式:首行缩进/悬挂缩进 + 输入数字 + 单位选择(字符/厘米/磅)
- 间距设置:
  - 段前:输入数字 + 单位选择(行/厘米/磅)
  - 段后:输入数字 + 单位选择(行/厘米/磅)
  - 行距:选择预定义选项(单倍行距/1.5 倍行距/2 倍行距/固定值/多倍行距) + 输入数字
- 预览区域:实时显示设置效果
- 确定/取消按钮

**界面要求:**
- 标题:段落
- 采用折叠式分组(缩进、间距)
- 输入框支持数字输入验证
- 单位选择下拉菜单
- 预览区域使用占位文本展示效果

#### 2.6.2 表格属性对话框

**功能要求:**
- 表格尺寸:
  - 表格宽度:输入数字 + 单位选择(厘米/磅/百分比)
  - 表格高度:输入数字 + 单位选择(厘米/磅)
- 列宽:
  - 均分列宽
  - 自定义各列宽度(输入框列表)
- 行高:
  - 均分行高
  - 自定义各行高度(输入框列表)
- 对齐方式:
  - 水平对齐:左对齐/居中/右对齐
  - 垂直对齐:顶端/居中/底端
- 边框:
  - 边框线型:实线/虚线/双线
  - 边框颜色选择
  - 边框粗细选择
- 背景:背景颜色选择
- 确定/取消按钮

**界面要求:**
- 标题:表格属性
- 采用分组布局(表格尺寸、列宽、行高、对齐方式、边框、背景)
- 输入框支持数字输入验证
- 下拉菜单和颜色选择器

#### 2.6.3 其他对话框
- 字体对话框:字体 family、大小、粗体、斜体、下划线、颜色等设置
- 图片格式对话框:大小、位置、环绕方式、旋转等设置
- 插入符号对话框:符号库浏览和插入

### 2.7 页面跳转功能

#### 2.7.1 核心功能
- 顶部跳转按钮:快速返回页面顶部
- 底部跳转按钮:快速滚动至页面底部
- 设置开关:用户可在设置中开启或关闭该功能

## 3. 菜单样式要求

### 3.1 菜单外观
- 蓝白配色风格,与 Microsoft Word 2010+ 一致
- 菜单背景:#ffffff
- 菜单项文字:#333333
- 悬停高亮背景:#e5f3ff
- 禁用状态文字:#a0a0a0
- 边框:1px solid #c0c0c0
- 分隔线:1px solid #e0e0e0
- 菜单圆角:2px
- 字体:'Microsoft YaHei', sans-serif
- 字号:12px

### 3.2 子菜单展开
- 子菜单位于父菜单右侧
- 子菜单保持与父菜单相同的样式
- 支持多级嵌套

### 3.3 对话框样式
- 对话框背景:#f0f0f0
- 标题栏背景:线性渐变(#ffffff 到 #e5e5e5)
- 关闭按钮:右上角 × 号
- 内容区域背景:#ffffff
- 分组标题背景:#f5f5f5
- 输入框边框:1px solid #a0a0a0
- 确定/取消按钮:线性渐变背景(#ffffff 到 #e5e5e5),悬停时蓝色高亮

## 4. 技术要求

### 4.1 实现方式
- 使用 HTML5 Canvas 作为绘制层
- 右键菜单和对话框使用 HTML/CSS 实现
- 事件监听:监听 canvas 的 contextmenu 事件,阻止默认行为,显示自定义菜单
- 位置计算:根据鼠标事件坐标和文档布局计算菜单显示位置
- 碰撞检测:确保菜单显示在视口内,必要时调整位置

### 4.2 交互细节
- 右键点击时显示菜单
- 点击菜单外区域关闭菜单
- 子菜单悬停显示
- 快捷键支持(Ctrl+C、Ctrl+V、Ctrl+X、Ctrl+A)
- 禁用不可用的菜单项

## 5. 输出要求

1. 完整的 ContextMenuManager 类代码
2. 段落对话框的 HTML/CSS 代码
3. 表格属性对话框的 HTML/CSS 代码
4. 右键菜单的 CSS 样式代码
5. 至少实现以下功能:
   - 段落右键:字体、段落对话框、项目符号、编号
   - 表格右键:插入行列、删除行列、合并单元格、拆分单元格、表格属性、9种对齐方式
   - 图片右键:环绕文字、大小位置
   - 页面跳转:顶部/底部跳转按钮及设置开关

## 6. 约束条件

- 禁止使用浏览器原生 contextmenu 事件以外的菜单实现方式
- 所有上下文必须使用同一个菜单系统进行动态切换
- 必须区分段落、表格、图片、空白区域的右键菜单
- 菜单项必须包含快捷键提示和下划线字母
- 数据层与渲染层保持分离,使用 JSON 描述文档结构
- 页面跳转功能需在设置中提供开启/关闭选项
