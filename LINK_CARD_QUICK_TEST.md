# 链接卡片功能快速测试指南

## 🚀 5分钟快速测试

### 测试1：插入文本链接（1分钟）

```
1. 在编辑器中输入："访问百度搜索"
2. 选中"百度搜索"四个字
3. 点击工具栏的"插入链接"按钮（链接图标）
4. 保持"文本链接"选项选中
5. 输入链接地址：https://www.baidu.com
6. 点击"插入链接"按钮
```

**预期结果**：
- ✅ 插入一个蓝色下划线的链接
- ✅ 链接文字是"百度搜索"
- ✅ 点击链接可以打开百度

---

### 测试2：插入基本卡片（2分钟）

```
1. 点击工具栏的"插入链接"按钮
2. 选择"卡片链接"选项
3. 输入链接地址：https://www.github.com
4. 输入卡片标题：GitHub
5. 点击"插入链接"按钮
```

**预期结果**：
- ✅ 插入一个白色背景的卡片
- ✅ 卡片显示标题"GitHub"
- ✅ 卡片底部显示URL
- ✅ 鼠标悬停时卡片有动画效果（向上移动、阴影加深、边框变蓝）
- ✅ 点击卡片可以打开GitHub

---

### 测试3：插入完整卡片（2分钟）

```
1. 点击工具栏的"插入链接"按钮
2. 选择"卡片链接"选项
3. 输入链接地址：https://www.google.com
4. 输入卡片标题：Google搜索
5. 输入卡片描述：全球最大的搜索引擎，提供网页、图片、视频等搜索服务
6. 输入图片URL：https://www.google.com/images/branding/googlelogo/2x/googlelogo_color_272x92dp.png
7. 点击"插入链接"按钮
```

**预期结果**：
- ✅ 插入一个带图片的卡片
- ✅ 左侧显示Google logo
- ✅ 右侧显示标题、描述和URL
- ✅ 鼠标悬停时有动画效果
- ✅ 点击卡片可以打开Google

---

## 📋 详细测试清单

### ✅ 测试4：卡片样式检查

**步骤**：
1. 插入一个完整卡片
2. 观察卡片的各个部分

**检查项**：
- [ ] 卡片有圆角边框
- [ ] 卡片有适当的内边距
- [ ] 图片是正方形（120x120px）
- [ ] 标题字体较大且加粗
- [ ] 描述文字颜色较浅
- [ ] URL文字最小且颜色最浅
- [ ] 整体布局美观

---

### ✅ 测试5：卡片悬停效果

**步骤**：
1. 插入一个卡片
2. 将鼠标移到卡片上
3. 观察动画效果

**检查项**：
- [ ] 卡片向上移动约2px
- [ ] 阴影变深
- [ ] 边框颜色变为蓝色（#4361ee）
- [ ] 过渡动画流畅（0.2秒）
- [ ] 鼠标移开后恢复原状

---

### ✅ 测试6：卡片点击功能

**步骤**：
1. 插入一个卡片
2. 点击卡片的不同位置

**检查项**：
- [ ] 点击标题可以打开链接
- [ ] 点击描述可以打开链接
- [ ] 点击图片可以打开链接
- [ ] 点击URL可以打开链接
- [ ] 链接在新标签页打开

---

### ✅ 测试7：导出HTML测试

**步骤**：
1. 插入2-3个不同的卡片（有图片和无图片）
2. 点击右侧"导出HTML"按钮
3. 在浏览器中打开导出的HTML文件
4. 测试卡片功能

**检查项**：
- [ ] 卡片样式完整保留
- [ ] 悬停效果正常
- [ ] 点击可以打开链接
- [ ] 图片正常显示
- [ ] 布局美观

---

### ✅ 测试8：移动端响应式测试

**步骤**：
1. 插入一个带图片的卡片
2. 导出为HTML
3. 在移动设备或浏览器开发者工具的移动模式下打开
4. 观察卡片布局

**检查项**：
- [ ] 卡片变为纵向布局
- [ ] 图片在上方，内容在下方
- [ ] 图片宽度100%
- [ ] 图片高度约200px
- [ ] 文字内容正常显示
- [ ] 点击功能正常

---

### ✅ 测试9：混合内容测试

**步骤**：
1. 创建一个包含以下内容的文档：
   - H1标题
   - 正文段落
   - 文本链接
   - 卡片链接（无图片）
   - 卡片链接（有图片）
   - 图片
   - 表格
2. 导出为HTML
3. 在浏览器中查看

**检查项**：
- [ ] 所有内容正常显示
- [ ] 卡片与其他内容间距合适
- [ ] 卡片不影响其他元素
- [ ] 整体布局协调

---


## 🐛 常见问题排查

### 问题1：卡片不显示
**检查**：
- [ ] 是否选择了"卡片链接"选项
- [ ] 是否填写了链接地址和标题
- [ ] 浏览器控制台是否有错误

### 问题2：卡片样式不正确
**检查**：
- [ ] 是否正确导出了HTML
- [ ] CSS样式是否正确加载
- [ ] 浏览器是否支持flexbox

### 问题3：图片不显示
**检查**：
- [ ] 图片URL是否正确
- [ ] 图片是否支持跨域访问
- [ ] 网络连接是否正常

### 问题4：点击卡片没有反应
**检查**：
- [ ] 是否在编辑器中（编辑器中卡片不可点击）
- [ ] 是否导出为HTML后测试
- [ ] 浏览器是否阻止了弹出窗口

### 问题5：悬停效果不流畅
**检查**：
- [ ] 浏览器性能是否正常
- [ ] 是否有其他CSS冲突
- [ ] 是否在导出的HTML中测试
