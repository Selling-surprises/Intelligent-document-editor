## 技术实现

### 选区保存和恢复机制

使用 `Range` API 保存和恢复用户选区：

1. **保存选区**：在打开对话框时，克隆当前选区的 Range 对象
2. **恢复选区**：在插入内容前，恢复之前保存的 Range 对象
3. **智能处理**：如果有选中文本，优先使用选中的文本作为链接文本

### JSON序列化

使用 `JSON.stringify` 和 `JSON.parse` 进行数据序列化：

1. **导出**：将内容和设置序列化为JSON字符串
2. **导入**：解析JSON字符串并验证数据格式
3. **错误处理**：捕获解析错误并显示友好的错误消息

### 文件下载和上传

使用 Blob API 和 FileReader API 实现文件操作
