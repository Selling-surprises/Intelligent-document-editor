# 缩进功能说明

## 📅 更新日期
2025-12-06

## 🎯 功能概述

为离线Word文档编辑器添加了增加/减少缩进功能，支持工具栏按钮和键盘快捷键两种操作方式。

---

## ✨ 功能特性

### 1. 工具栏按钮

**位置**：工具栏中，位于列表按钮和对齐按钮之间

**按钮**：
- 📥 **增加缩进**：向右缩进选中的文本或段落
- 📤 **减少缩进**：向左减少缩进

**图标**：
- 增加缩进：使用`IndentIncrease`图标（向右箭头）
- 减少缩进：使用`IndentDecrease`图标（向左箭头）

### 2. 键盘快捷键

**快捷键**：
- `Tab`：增加缩进
- `Shift + Tab`：减少缩进

**优势**：
- ✅ 快速操作，无需移动鼠标
- ✅ 符合主流文本编辑器的习惯
- ✅ 提高编辑效率

---

## 🎨 使用场景

### 场景1：段落缩进

**操作**：
1. 选中需要缩进的段落
2. 点击"增加缩进"按钮或按`Tab`键
3. 段落向右缩进

**效果**：
```
原始：
这是第一段。
这是第二段。

缩进后：
这是第一段。
    这是第二段。
```

### 场景2：列表层级

**操作**：
1. 创建一个无序列表或有序列表
2. 选中某个列表项
3. 点击"增加缩进"按钮或按`Tab`键
4. 列表项变为子列表

**效果**：
```
原始：
• 项目1
• 项目2
• 项目3

缩进后：
• 项目1
    • 项目2
• 项目3
```

### 场景3：代码块缩进

**操作**：
1. 选中代码块中的某几行
2. 按`Tab`键增加缩进
3. 按`Shift + Tab`减少缩进

**效果**：
```javascript
// 原始
function hello() {
console.log("Hello");
}

// 缩进后
function hello() {
    console.log("Hello");
}
```

### 场景4：引用文本

**操作**：
1. 选中需要引用的文本
2. 多次点击"增加缩进"按钮
3. 创建多层级引用效果

**效果**：
```
原始文本

    一级引用

        二级引用
```

## 🔧 技术实现

### 1. 工具栏按钮

**文件**：`src/components/editor/EditorToolbar.tsx`

**实现**：
```tsx
// 导入图标
import {
  IndentIncrease,
  IndentDecrease,
} from 'lucide-react';

// 添加按钮
<div className="flex items-center gap-1">
  <Button
    variant="ghost"
    size="icon"
    onClick={() => onCommand('indent')}
    title="增加缩进"
  >
    <IndentIncrease className="h-4 w-4" />
  </Button>
  <Button
    variant="ghost"
    size="icon"
    onClick={() => onCommand('outdent')}
    title="减少缩进"
  >
    <IndentDecrease className="h-4 w-4" />
  </Button>
</div>
```

### 2. 键盘快捷键

**文件**：`src/components/editor/EditorContent.tsx`

**实现**：
```tsx
const handleKeyDown = (e: React.KeyboardEvent) => {
  // Tab键：增加缩进
  if (e.key === 'Tab' && !e.shiftKey) {
    e.preventDefault();
    document.execCommand('indent', false);
  }
  // Shift+Tab：减少缩进
  else if (e.key === 'Tab' && e.shiftKey) {
    e.preventDefault();
    document.execCommand('outdent', false);
  }
};
```

### 3. execCommand API

**使用的命令**：
- `document.execCommand('indent', false)`：增加缩进
- `document.execCommand('outdent', false)`：减少缩进

**浏览器支持**：
- ✅ Chrome
- ✅ Firefox
- ✅ Safari
- ✅ Edge
