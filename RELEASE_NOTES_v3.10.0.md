# 离线Word文档编辑器 v3.10.0 发布说明

发布日期：2025-12-06

## 🎉 重大更新

本次更新带来了多项重要功能，大幅提升了文档编辑体验！

### ✨ 核心新功能

#### 1. 表格操作工具栏
插入表格后，现在可以轻松地对表格进行各种操作：
- **添加行/列**：快速扩展表格
- **删除行/列**：精确删除不需要的内容
- **删除整个表格**：一键清除
- **智能显示**：选中表格时自动显示工具栏，不选中时自动隐藏

#### 2. 代码插入功能
为技术文档编写者提供专业的代码插入功能：
- 支持20+种编程语言
- 自动语法高亮
- 专业的代码块样式
- 支持的语言包括：JavaScript、Python、Java、C++、Go、Rust、TypeScript、HTML、CSS、SQL等

#### 3. 文本对齐工具
新增完整的文本对齐功能：
- 左对齐
- 居中对齐
- 右对齐
- 两端对齐

#### 4. 悬浮目录导出
导出的HTML文件现在自动包含智能悬浮目录：
- 默认隐藏在左侧边缘
- 鼠标悬停自动展开
- 可固定显示/隐藏
- 点击目录项平滑滚动到对应位置
- 完美适配移动端

### 🐛 问题修复
- 修复了文件选择按钮中文字垂直对齐问题

### 🎨 界面优化
- 移除了工具栏中的插入目录按钮（导出时自动生成）
- 优化了表格工具栏的视觉设计

## 🚀 使用建议

### 表格编辑工作流
1. 使用表格插入功能创建初始表格
2. 点击表格任意位置激活表格工具栏
3. 使用工具栏按钮调整表格结构
4. 完成后点击表格外部区域隐藏工具栏

### 代码文档编写
1. 编写技术文档时，点击代码图标
2. 选择对应的编程语言
3. 粘贴或输入代码
4. 代码会自动应用语法高亮

### 专业文档导出
1. 使用H1、H2、H3标题组织文档结构
2. 使用对齐工具美化文本布局
3. 导出HTML文件
4. 导出的文件自动包含悬浮目录，方便阅读

## 🔧 技术细节

- **代码高亮库**：highlight.js
- **支持的编程语言**：20+种主流语言
- **表格操作**：基于DOM操作的实时编辑
- **目录生成**：自动识别H1-H3标题
- **悬浮目录**：纯CSS+JavaScript实现，无需额外依赖
