## 🎯 关于目录功能

### 目录功能是如何工作的？

当您在文档中添加**标题**（H1、H2、H3）后，导出的HTML文件会自动包含一个悬浮目录。

### ⚠️ 重要：什么是"标题"？

**正确的做法**：
1. 选中文字
2. 点击工具栏的**"标题"下拉菜单**
3. 选择"标题 1"、"标题 2"或"标题 3"
4. 文字会变大且变成蓝色

**错误的做法**：
- ❌ 只是把文字加粗
- ❌ 只是把文字放大
- ❌ 只是改变文字颜色

### 如何查看导出HTML的目录？

目录默认隐藏在页面左侧，有两种方式显示：

1. **鼠标悬停**：将鼠标移到页面最左侧边缘
2. **点击按钮**：点击蓝色的"☰"按钮（有光晕闪烁）

---

## 🔍 如何诊断问题？

### 步骤1：导出时查看控制台

1. 按 **F12** 打开浏览器控制台
2. 点击 **导出** 按钮
3. 查看控制台输出

### 步骤2：检查关键信息

查看控制台中的这些信息：

```
找到的标题数量: X
```

- **如果是 0**：说明您没有正确添加标题
  - 解决方法：使用工具栏的"标题"下拉菜单添加标题
  
- **如果大于 0**：说明标题已提取，目录已生成
  - 解决方法：打开HTML文件，将鼠标移到左侧边缘查看目录

### 步骤3：查看详细输出

控制台会显示：
- 每个标题的文本和层级
- 是否生成目录
- 生成的目录HTML代码

## 💡 快速测试

想快速测试目录功能？按照以下步骤：

1. **清空编辑器**
2. **输入以下内容**：
   ```
   第一章
   这是第一章的内容
   
   1.1 简介
   这是简介的内容
   
   第二章
   这是第二章的内容
   ```
3. **设置标题**：
   - 选中"第一章"，设置为"标题 1"
   - 选中"1.1 简介"，设置为"标题 2"
   - 选中"第二章"，设置为"标题 1"
4. **导出并查看**：
   - 按F12打开控制台
   - 点击导出
   - 查看控制台输出（应该显示"找到的标题数量: 3"）
   - 打开HTML文件
   - 将鼠标移到左侧边缘
