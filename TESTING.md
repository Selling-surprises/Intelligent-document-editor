# 功能测试指南

## v5.1.0 新功能测试

### 1. JSON导入导出功能测试

#### 测试步骤：

**导出JSON**
1. 在编辑器中输入一些内容（文字、表格、链接等）
2. 点击右上角的"设置"按钮（齿轮图标）
3. 滚动到设置面板底部
4. 点击"导出JSON"按钮
5. 验证：应该下载一个JSON文件，文件名为"[页面标题].json"

**导入JSON**
1. 清空编辑器内容（或刷新页面）
2. 点击右上角的"设置"按钮
3. 点击"导入JSON"按钮
4. 选择之前导出的JSON文件
5. 验证：
   - 编辑器内容应该恢复
   - 所有设置（页面标题、图标、背景等）应该恢复
   - 应该显示"导入成功"的提示

**JSON文件格式验证**
打开导出的JSON文件，应该包含：
```json
{
  "version": "5.1.0",
  "content": "...",
  "settings": {
    "pageTitle": "...",
    "favicon": "...",
    "backgroundImage": "...",
    "opacity": ...
  },
  "exportDate": "2025-12-06T..."
}
```

### 2. 链接插入功能测试

#### 测试场景1：选中文字后插入链接

1. 在编辑器中输入文字："点击这里访问网站"
2. 用鼠标选中"点击这里"这几个字
3. 点击工具栏的链接按钮（链条图标）
4. 在弹出的对话框中输入URL：https://example.com
5. 点击"插入链接"
6. 验证：
   - "点击这里"应该变成蓝色链接
   - 链接文本应该是"点击这里"
   - 点击链接应该能打开 https://example.com

#### 测试场景2：未选中文字插入链接

1. 将光标放在编辑器中的任意位置
2. 点击工具栏的链接按钮
3. 输入URL：https://example.com
4. 输入显示文本：示例网站
5. 点击"插入链接"
6. 验证：
   - 应该插入一个"示例网站"的链接
   - 链接应该指向 https://example.com

#### 测试场景3：只输入URL不输入文本

1. 点击工具栏的链接按钮
2. 只输入URL：https://example.com
3. 不输入显示文本
4. 点击"插入链接"
5. 验证：
   - 应该插入一个链接
   - 链接文本应该是URL本身：https://example.com

### 3. 表格边框样式测试

#### 测试步骤：

**插入表格**
1. 点击工具栏的表格按钮
2. 选择3行3列
3. 点击"插入表格"
4. 验证：
   - 表格应该有明显的2px实线边框
   - 边框颜色应该是深色（#333）
   - 应该显示"表格插入成功"的提示

**修改边框粗细**
1. 右键点击表格中的任意单元格
2. 选择"边框样式" → "边框粗细"
3. 依次测试：
   - 细线（1px）：边框应该变细
   - 中等（2px）：边框应该是中等粗细
   - 粗线（3px）：边框应该变粗

**修改边框样式**
1. 右键点击表格中的任意单元格
2. 选择"边框样式" → "边框样式"
3. 依次测试：
   - 实线：边框应该是实线
   - 虚线：边框应该是虚线
   - 点线：边框应该是点线
   - 双线：边框应该是双线

### 4. 综合测试

#### 完整工作流测试

1. 创建一个包含以下内容的文档：
   - 标题（H1）
   - 段落文字
   - 一个3x3的表格，设置为虚线边框
   - 一个链接（选中文字后插入）
   - 一张图片

2. 设置页面：
   - 修改页面标题为"测试文档"
   - 上传一个favicon
   - 设置背景图片
   - 调整透明度为80%

3. 导出JSON：
   - 点击"导出JSON"
   - 验证文件下载成功

4. 清空并导入：
   - 刷新页面（清空内容）
   - 点击"导入JSON"
   - 选择刚才导出的文件
   - 验证所有内容和设置都恢复了

5. 导出HTML：
   - 点击"导出为HTML文件"
   - 在浏览器中打开HTML文件
   - 验证所有内容正确显示
