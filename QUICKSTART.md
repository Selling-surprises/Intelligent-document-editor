# 快速开始指南

## 基本使用

### 1. 文字编辑

- **格式化文字**：选中文字后，使用工具栏的按钮进行加粗、斜体、下划线等操作
- **修改字体**：在工具栏选择字体下拉菜单
- **修改字号**：在工具栏选择字号下拉菜单
- **修改颜色**：点击文字颜色或背景色按钮，选择颜色

### 2. 插入链接

#### 方法一：选中文字后插入（推荐）

1. 用鼠标选中要添加链接的文字
2. 点击工具栏的链接按钮（🔗）
3. 在弹出的对话框中输入链接地址
4. 点击"插入链接"

#### 方法二：直接插入

1. 将光标放在要插入链接的位置
2. 点击工具栏的链接按钮
3. 输入链接地址和显示文本
4. 点击"插入链接"

### 3. 插入表格

1. 点击工具栏的表格按钮
2. 选择行数和列数
3. 点击"插入表格"
4. 右键点击表格可以：
   - 添加/删除行列
   - 合并/拆分单元格
   - 设置边框样式

#### 自定义表格边框

1. 右键点击表格中的任意单元格
2. 选择"边框样式"
3. 选择边框粗细：
   - 细线（1px）
   - 中等（2px）- 默认
   - 粗线（3px）
4. 选择边框样式：
   - 实线 - 标准表格
   - 虚线 - 草稿表格
   - 点线 - 辅助表格
   - 双线 - 强调表格

### 4. JSON导入导出

#### 导出JSON（保存工作）

1. 点击右上角的"设置"按钮（⚙️）
2. 滚动到设置面板底部
3. 点击"导出JSON"按钮
4. JSON文件会自动下载

**JSON文件包含：**
- 文档内容（包括文字、表格、链接等）
- 页面设置（标题、图标、背景图片、透明度）
- 版本信息和导出时间

#### 导入JSON（恢复工作）

1. 点击右上角的"设置"按钮
2. 点击"导入JSON"按钮
3. 选择之前导出的JSON文件
4. 所有内容和设置会立即恢复

**使用场景：**
- 保存编辑进度，下次继续编辑
- 在不同设备间迁移文档
- 备份重要文档
- 分享文档给他人编辑

### 5. 导出HTML

1. 点击右上角的"设置"按钮
2. 点击"导出为HTML文件"按钮
3. HTML文件会自动下载
4. 用浏览器打开HTML文件即可查看

**HTML文件特性：**
- 独立的HTML文件，无需网络即可查看
- 自动生成悬浮目录（如果有H1-H3标题）
- 保留所有格式和样式
- 可以直接分享给他人查看

### 6. 页面设置

点击右上角的"设置"按钮，可以设置：

- **页面标题**：修改文档标题
- **页面图标**：上传favicon或输入图标URL
- **背景图片**：上传背景图片或输入图片URL
- **透明度**：调整背景图片的透明度

### 7. 撤销和重做

- **撤销**：点击工具栏的撤销按钮（↶）或按 Ctrl+Z
- **重做**：点击工具栏的重做按钮（↷）或按 Ctrl+Y

## 高级功能

### 插入多媒体

- **图片**：点击图片按钮，输入图片URL
- **视频**：点击视频按钮，输入视频URL（支持YouTube、Bilibili等）
- **音频**：点击音频按钮，输入音频URL（支持网易云音乐、QQ音乐等）

### 插入代码

1. 点击工具栏的代码按钮（</>）
2. 输入代码
3. 选择编程语言
4. 点击"插入代码"

支持的语言：JavaScript、Python、Java、C++、HTML、CSS等

## 快捷键

- **Ctrl+B**：加粗
- **Ctrl+I**：斜体
- **Ctrl+U**：下划线
- **Ctrl+Z**：撤销
- **Ctrl+Y**：重做

## 常见问题

### Q: JSON文件和HTML文件有什么区别？

- **JSON文件**：用于保存和恢复编辑状态，可以继续编辑
- **HTML文件**：用于查看和分享，不能编辑

### Q: 如何备份我的文档？

定期导出JSON文件即可。建议：
1. 编辑过程中定期导出JSON
2. 完成后导出HTML用于查看和分享

### Q: 链接插入后无法点击？

在编辑模式下，链接不可点击。导出为HTML后，链接就可以正常点击了。

### Q: 表格边框看不清？

1. 右键点击表格
2. 选择"边框样式" → "边框粗细" → "粗线（3px）"
3. 或者选择"边框样式" → "边框样式" → "双线"

### Q: 如何在不同设备间同步文档？

1. 在设备A上导出JSON文件
2. 将JSON文件传输到设备B（通过邮件、云盘等）
3. 在设备B上导入JSON文件

## 技巧和建议

1. **定期保存**：养成定期导出JSON的习惯，避免意外丢失内容
2. **使用标题**：使用H1-H3标题，导出HTML时会自动生成目录
3. **表格边框**：插入表格后，使用右键菜单设置合适的边框样式
4. **选中文字插入链接**：先选中文字再插入链接，更加方便
5. **背景图片**：使用浅色背景图片，调整透明度到80-90%，效果更好
