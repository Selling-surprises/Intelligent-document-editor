## 🔍 如何验证右键菜单是否工作

### 方法 1: 在编辑器中测试

1. **打开应用**
   - 访问编辑器页面
   - 等待页面完全加载

2. **测试段落右键菜单**
   - 在编辑器中输入一些文字
   - 右键点击文字
   - 应该看到自定义的右键菜单（不是浏览器默认菜单）

3. **测试表格右键菜单**
   - 使用工具栏插入一个表格
   - 右键点击表格单元格
   - 应该看到表格专用的右键菜单

4. **测试图片右键菜单**
   - 使用工具栏插入一张图片
   - 右键点击图片
   - 应该看到图片专用的右键菜单

### 方法 2: 检查浏览器控制台

1. **打开开发者工具**
   - 按 F12 或右键点击页面选择"检查"
   - 切换到 Console 标签

2. **查看日志**
   - 右键点击编辑器内容
   - 应该看到类似这样的日志：
     ```
     执行右键菜单命令: [命令名] [参数] [上下文]
     ```

3. **检查错误**
   - 如果有红色错误信息，说明有问题
   - 如果没有错误，说明右键菜单正常工作

## 🐛 可能的问题和解决方案

### 问题 1: 右键菜单仍然不显示

**可能原因**：
- 编辑器元素的 ref 没有正确绑定
- 事件监听器没有正确添加

**解决方案**：
1. 检查浏览器控制台是否有错误
2. 确认 editorContentRef 是否正确获取到 DOM 元素
3. 尝试刷新页面

**调试代码**：
```tsx
// 在 useEffect 中添加调试日志
useEffect(() => {
  if (editorRef.current) {
    const element = editorRef.current.getElement();
    console.log('编辑器元素:', element);
    if (element) {
      editorContentRef.current = element;
      console.log('editorContentRef 已设置:', editorContentRef.current);
    }
  }
}, []);
```

### 问题 2: 右键菜单显示但位置不对

**可能原因**：
- CSS z-index 冲突
- 父容器有 transform 属性影响 fixed 定位

**解决方案**：
1. 检查菜单的 z-index 是否足够高（默认 10000）
2. 检查父容器是否有 transform 属性
3. 菜单组件会自动进行边界检测和位置调整

### 问题 3: 点击菜单项没有反应

**可能原因**：
- 命令处理器没有正确实现
- 事件冒泡被阻止

**解决方案**：
1. 检查 onCommand 回调是否正确执行
2. 查看浏览器控制台的日志
3. 确认命令名称是否正确

### 问题 4: 对话框不显示

**可能原因**：
- 对话框状态没有正确更新
- 对话框组件没有正确渲染

**解决方案**：
1. 检查 paragraphDialogOpen 和 tablePropertiesDialogOpen 状态
2. 确认对话框组件已正确导入
3. 查看浏览器控制台是否有错误

## 🔧 调试技巧

### 1. 添加调试日志

在 useContextMenu Hook 中添加日志：

```tsx
const {
  menuState,
  closeMenu,
  executeCommand,
  // ...
} = useContextMenu({
  editorRef: editorContentRef as React.RefObject<HTMLDivElement>,
  onCommand: (command, item, context) => {
    console.log('🎯 右键菜单命令:', {
      command,
      item,
      context,
      editorRef: editorContentRef.current
    });
    // ...
  }
});
```

### 2. 检查事件监听器

在浏览器控制台中运行：

```javascript
// 检查编辑器元素
const editor = document.querySelector('.editor-content');
console.log('编辑器元素:', editor);

// 检查事件监听器
getEventListeners(editor);
```

### 3. 测试上下文检测

在浏览器控制台中运行：

```javascript
// 模拟右键点击
const editor = document.querySelector('.editor-content');
const event = new MouseEvent('contextmenu', {
  bubbles: true,
  cancelable: true,
  clientX: 100,
  clientY: 100
});
editor.dispatchEvent(event);
```
