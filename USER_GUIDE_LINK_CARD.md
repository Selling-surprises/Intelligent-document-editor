# 链接卡片完整使用指南

## 🎯 功能概述

链接卡片是一种精美的链接展示方式，支持显示标题、描述和图片，并提供编辑和删除功能。

---

## 📖 使用流程

### 1️⃣ 插入链接卡片

#### 步骤1：打开插入对话框
点击工具栏的 **🔗 插入链接** 按钮

#### 步骤2：选择卡片类型
选择 **卡片链接（精美卡片样式）**

#### 步骤3：输入链接地址
在"链接地址"输入框中输入网址

#### 步骤4：自动获取信息（推荐）
点击 **✨ 自动获取** 按钮，系统会自动填充：
- 📝 卡片标题
- 📄 卡片描述
- 🖼️ 卡片图片

#### 步骤5：手动调整（可选）
如果自动获取的信息不准确，可以手动修改

#### 步骤6：插入卡片
点击 **插入链接** 按钮完成

---

### 2️⃣ 编辑链接卡片

#### 步骤1：悬停显示按钮
将鼠标移到已插入的链接卡片上，右上角会显示 **编辑** 和 **删除** 按钮

#### 步骤2：点击编辑按钮
点击蓝色的 **编辑** 按钮

#### 步骤3：修改信息
依次输入新的：
- 链接地址
- 标题
- 描述（可选）

#### 步骤4：确认修改
点击"确定"按钮，卡片内容会立即更新

---

### 3️⃣ 删除链接卡片

#### 步骤1：悬停显示按钮
将鼠标移到已插入的链接卡片上

#### 步骤2：点击删除按钮
点击红色的 **删除** 按钮

#### 步骤3：确认删除
在确认对话框中点击"确定"

#### 步骤4：卡片被删除
卡片会立即从文档中移除

---

## 💡 使用技巧

### 技巧1：快速创建
使用自动获取功能，创建速度提升5-10倍：
```
输入链接 → 点击自动获取 → 等待1-3秒 → 插入
```

### 技巧2：批量插入
对多个链接重复以下步骤：
```
1. 输入链接地址
2. 点击自动获取
3. 插入链接
4. 重复
```

### 技巧3：快速修正
发现链接地址错误时：
```
悬停 → 点击编辑 → 修改链接 → 确认
```

### 技巧4：撤销操作
误删卡片时：
```
按 Ctrl+Z（Windows）或 Cmd+Z（Mac）撤销
```

## 🎨 视觉效果

### 默认状态
卡片以精美的样式显示，包含：
- 左侧图片（如果有）
- 标题（粗体显示）
- 描述（灰色文字）
- 链接地址（小字显示）

### 悬停状态
鼠标悬停时：
- 卡片轻微上浮
- 显示阴影效果
- 右上角显示编辑和删除按钮
- 边框颜色变为蓝色

### 点击效果
- 点击卡片内容：在新标签页打开链接
- 点击编辑按钮：弹出编辑对话框
- 点击删除按钮：删除卡片

## ⚠️ 注意事项

### 自动获取功能

**成功率**：
- ✅ 大多数网站支持（约80-90%）
- ⚠️ 部分网站可能限制爬取
- ⚠️ 某些网站没有设置元数据

**失败时的处理**：
1. 手动填写标题和描述
2. 图片URL可以留空
3. 不影响卡片插入

### 编辑和删除

**编辑器内**：
- ✅ 编辑和删除会触发内容更新
- ✅ 可以使用撤销功能恢复
- ✅ 会显示Toast提示

**导出HTML**：
- ⚠️ 编辑后需要重新保存HTML文件
- ⚠️ 浏览器不会自动保存修改
- ⚠️ 删除后无法撤销（除非刷新页面）

## 🔧 常见问题

### Q1：为什么看不到编辑和删除按钮？

**A：** 可能的原因：
1. 鼠标没有悬停在卡片上
2. 卡片是旧版本创建的（需要重新插入）
3. 浏览器缓存问题（刷新页面）

**解决方法**：
- 确保鼠标悬停在卡片上
- 重新插入链接卡片
- 刷新页面

---

### Q2：自动获取失败怎么办？

**A：** 可能的原因：
1. 网站没有设置元数据
2. 网站限制爬取
3. 网络问题

**解决方法**：
- 手动填写标题和描述
- 稍后重试
- 使用其他链接

---

### Q3：编辑后内容没有更新？

**A：** 可能的原因：
1. 点击了取消按钮
2. 输入的内容为空
3. 浏览器阻止了弹出对话框

**解决方法**：
- 确认点击了"确定"按钮
- 确保输入了有效内容
- 检查浏览器设置

---

### Q4：删除后能恢复吗？

**A：** 
- **编辑器内**：可以使用撤销功能（Ctrl+Z）恢复
- **导出HTML**：无法恢复，除非刷新页面

---

### Q5：如何批量删除卡片？

**A：** 
- 逐个悬停并点击删除
- 或者选中包含卡片的区域，按Delete键

## 🎯 最佳实践

### 推荐做法

1. **优先使用自动获取**
   - 节省时间
   - 减少错误
   - 获取准确信息

2. **检查自动获取的内容**
   - 确认标题准确
   - 确认描述相关
   - 确认图片合适

3. **及时修正错误**
   - 发现错误立即编辑
   - 不要等到导出后再修改

4. **定期保存文档**
   - 使用JSON导出保存
   - 避免数据丢失

### 不推荐做法

1. **完全依赖自动获取**
   - 某些网站可能获取失败
   - 需要手动检查

2. **不检查内容就插入**
   - 可能插入错误信息
   - 影响文档质量

3. **在导出HTML后编辑**
   - 需要重新保存
   - 容易丢失修改
