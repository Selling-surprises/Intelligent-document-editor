## ✨ 主要功能改进

### 1. 表格列选择功能 ⭐⭐⭐⭐⭐

**功能描述**：
- 点击表格单元格时，整列自动高亮显示
- 高亮颜色：浅蓝色背景（rgba(67, 97, 238, 0.1)）
- 切换单元格时，高亮自动更新

**操作改进**：
- ✅ 删除列：基于当前高亮的列
- ✅ 清空列：一键清空整列所有单元格
- ✅ 视觉反馈：清楚地知道当前操作的是哪一列

**使用场景**：
```
点击这里 ↓
┌─────────┬─────────┬─────────┐
│  姓名   │  年龄   │  城市   │
├─────────┼─────────┼─────────┤
│  张三   │ [高亮]  │  北京   │
├─────────┼─────────┼─────────┤
│  李四   │ [高亮]  │  上海   │
└─────────┴─────────┴─────────┘
         ↑ 整列高亮
```

---

### 2. 导出HTML目录交互优化 ⭐⭐⭐⭐⭐

**v3.11.0改进**：
- 从鼠标悬停改为点击图标
- 左侧固定显示目录图标按钮（☰）
- 点击按钮显示/隐藏目录面板

**v3.11.1改进**：
- 点击目录按钮后，按钮会暂时消失
- 目录面板右上角显示折叠按钮（✕）
- 点击折叠按钮，目录隐藏，目录按钮重新出现
- 避免两个按钮同时显示，视觉更清晰

**交互流程**：
```
[☰] 点击 → [目录面板 ✕] 点击✕ → [☰]
```

## 📝 技术亮点

### 1. 表格列选择
- 使用`data-column-selected`属性标记高亮单元格
- 使用`selectedColumn`状态管理当前选中的列
- 自动清除旧的高亮并应用新的高亮

### 2. 目录交互
- 使用`.hidden`类控制按钮显示/隐藏
- 使用`opacity`和`pointer-events`实现平滑过渡
- 折叠按钮使用绝对定位，位于面板右上角

### 3. 背景图片
- 使用`background-attachment: fixed`固定背景
- 编辑区使用`rgba`实现半透明效果
- 导出HTML采用相同的背景处理方式

### 4. 响应式设计
- 工具栏位置根据表格位置动态计算
- 目录面板使用`transform`实现滑动效果
- 设置面板使用`max-h-[calc(100vh-120px)]`自适应高度
