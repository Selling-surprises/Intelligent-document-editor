# 毛玻璃效果快速上手指南

## 🚀 5分钟快速上手

### 步骤1：开启毛玻璃效果（30秒）

1. 打开编辑器
2. 在右侧设置面板找到"背景设置"
3. 向下滚动找到"启用毛玻璃效果"
4. 点击开关，开启效果

✅ **完成！** 您已经成功开启毛玻璃效果。

---

### 步骤2：调整模糊程度（1分钟）

1. 在"启用毛玻璃效果"下方会出现"模糊程度"滑块
2. 拖动滑块调整数值（推荐：15px）
3. 实时观察编辑器效果
4. 找到您喜欢的模糊程度

✅ **完成！** 您已经调整好模糊程度。

---

### 步骤3：配合背景图片（2分钟）

1. 在"背景设置"中找到"背景图片"
2. 点击"选择文件"或拖拽图片上传
3. 调整"内容透明度"到70%
4. 观察毛玻璃效果

✅ **完成！** 您已经创建了完美的视觉效果。

---

### 步骤4：导出HTML（1分钟）

1. 在编辑器中输入一些内容
2. 点击"导出为HTML文件"按钮
3. 保存HTML文件
4. 在浏览器中打开查看效果

✅ **完成！** 您的文档已经导出，毛玻璃效果完整保留。

---

## 💡 快速技巧

### 技巧1：快速切换效果

- 点击开关即可快速开启/关闭
- 无需重新设置模糊程度

### 技巧2：找到最佳模糊值

- 轻微模糊：8-12px
- 中等模糊：13-18px
- 强烈模糊：19-25px

### 技巧3：配合透明度

- 透明度高（80-90%）+ 低模糊（8-12px）= 清爽
- 透明度中（65-75%）+ 中模糊（15-20px）= 平衡
- 透明度低（50-60%）+ 高模糊（20-25px）= 艺术

## 🎯 常见问题

### Q：看不到毛玻璃效果？

**A**：检查三点：
1. 开关是否开启（蓝色）
2. 模糊程度是否大于0
3. 透明度是否小于100%

### Q：效果太弱？

**A**：
- 增加模糊程度（18-25px）
- 降低透明度（60-70%）
- 上传背景图片

### Q：效果太强？

**A**：
- 降低模糊程度（8-12px）
- 提高透明度（80-90%）
