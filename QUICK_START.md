# 离线Word文档编辑器 - 快速开始

## 🚀 5分钟上手指南

### 第一步：了解界面

打开编辑器后，您会看到：

```
┌─────────────────────────────────────────┐
│  离线Word文档编辑器 v3.9                 │
├─────────────────────────────────────────┤
│  [工具栏] 撤销 重做 | 字体 字号 | ...   │
├──────┬──────────────────────┬───────────┤
│ 目录 │   白色文档编辑区域    │  设置面板  │
│ 面板 │   （A4纸张大小）      │           │
├──────┴──────────────────────┴───────────┤
│  字符数: 0 | 字数: 0                     │
└─────────────────────────────────────────┘
```

---

### 第二步：开始编辑

#### 1. 输入标题
1. 在编辑区域输入文字
2. 选中文字
3. 在工具栏选择"标题 1"
4. 标题会自动出现在左侧目录中

#### 2. 格式化文本
- **加粗**：选中文字，点击 **B** 按钮
- **斜体**：选中文字，点击 *I* 按钮
- **颜色**：选中文字，点击调色板图标，选择颜色

#### 3. 插入表格 ⭐ 新功能
1. 点击工具栏中的表格图标（📊）
2. 设置行数和列数（如：3行4列）
3. 点击"插入表格"
4. 在单元格中输入内容

#### 4. 插入图片
**本地图片**：
1. 点击图片图标
2. 选择"本地上传"
3. 选择图片文件

**在线图片** ⭐ 新功能：
1. 点击图片图标
2. 选择"在线图片"
3. 输入图片URL
4. 点击"插入图片"

---

### 第三步：美化文档

#### 设置背景图片
1. 打开右侧设置面板
2. 在"背景图片"部分选择图片
3. 调整透明度滑块（建议60-80%）

#### 自定义标题
1. 在设置面板找到"网页标题"
2. 输入您的文档标题
3. 标题会影响导出的文件名

---

### 第四步：导出文档

1. 点击设置面板底部的"导出为HTML文件"按钮
2. 文件会自动下载
3. 双击打开HTML文件即可查看

## ⚡ 高效技巧

### 1. 使用标题生成目录
- 使用H1作为文档总标题
- 使用H2作为章节标题
- 使用H3作为小节标题
- 目录会自动生成，点击可跳转

### 2. 颜色使用建议
- **红色**：警告、错误、重要提示
- **黄色高亮**：关键信息、重点内容
- **蓝色**：链接、标题、强调
- **绿色**：成功、完成、正确

### 3. 表格使用技巧
- 先规划好行列数再插入
- 使用加粗突出表头
- 使用颜色区分不同类型的数据
- 保持表格简洁，避免过于复杂

### 4. 在线图片技巧
- 使用稳定的图床服务
- 确保图片URL可长期访问
- 测试URL是否有效
- 重要图片建议保存本地副本

## 🔧 常见问题

### Q: 如何保存文档？
A: 编辑器会自动保存历史记录（最多50步）。要永久保存，请点击"导出为HTML文件"。

### Q: 导出的HTML文件可以编辑吗？
A: 导出的HTML文件是静态的，不能直接编辑。如需修改，请在编辑器中修改后重新导出。

### Q: 图片会保存在哪里？
A: 本地上传的图片会转换为Base64编码，直接嵌入HTML文件中。在线图片则保持URL引用。

### Q: 表格可以合并单元格吗？
A: 当前版本不支持合并单元格，但可以手动调整表格内容。

### Q: 支持哪些浏览器？
A: 支持Chrome、Firefox、Safari、Edge等现代浏览器（版本要求见文档）。

### Q: 数据安全吗？
A: 所有数据仅在浏览器本地处理，不会上传到任何服务器，完全安全。

## 🌟 最佳实践

### 文档结构
```
标题 1：文档总标题
  标题 2：第一章
    标题 3：第一节
    正文内容
    表格、图片、列表
  标题 2：第二章
    标题 3：第一节
    正文内容
```

### 格式规范
- 标题使用H1-H3，不要跳级
- 段落之间保持适当间距
- 列表项保持简洁
- 表格数据对齐一致
- 图片大小适中

### 颜色搭配
- 主色调：蓝色（标题、链接）
- 强调色：红色（重要提示）
- 辅助色：黄色（高亮）、绿色（成功）
- 避免过多颜色，保持简洁
