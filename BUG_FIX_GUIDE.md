## 📝 使用建议

### 表格边框样式
如果需要设置表格边框：
1. 右键点击表格 → 表格属性
2. 切换到"边框"选项卡
3. **向下滚动**查看所有选项
4. 选择边框样式类型
5. 调整边框宽度、线型和颜色
6. 点击"应用"

### 首行缩进
如果需要设置首行缩进：
1. 选中段落或将光标放在段落中
2. 右键 → 段落 → 段落设置
3. 在"特殊格式"中选择"首行缩进"
4. 输入缩进值（建议2字符）
5. 点击"应用"

### 选区操作
如果需要对选中的文字应用段落设置：
1. 先选中文字
2. 右键 → 段落 → 段落设置
3. 修改设置
4. 点击"应用"
5. 选中状态会自动保持
