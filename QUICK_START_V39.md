# v3.9 快速上手指南

## 🚀 5分钟快速体验

### 1️⃣ 实时格式检测（1分钟）

```
1. 输入一些文字
2. 选中文字，设置为"标题 1"
3. 点击标题文字
4. 👀 看工具栏：标题选择器显示"标题 1"
5. 点击其他正文
6. 👀 看工具栏：标题选择器显示"正文"
```

**效果**：工具栏实时显示当前格式状态！

---

### 2️⃣ 智能链接插入（1分钟）

```
1. 输入："访问百度"
2. 选中"百度"两个字
3. 点击链接按钮
4. 输入：https://www.baidu.com
5. 点击插入
6. 👀 看结果：链接文本是"百度"
```

**效果**：自动使用选中的文字作为链接文本！

---

### 3️⃣ 图片查看器（3分钟）

```
1. 点击"插入图片"按钮
2. 上传2张图片
3. 点击右侧"导出HTML"按钮
4. 在浏览器中打开导出的HTML文件
5. 点击任意图片
6. 👀 看效果：图片全屏显示
7. 点击"+"按钮放大
8. 点击"−"按钮缩小
9. 按ESC键关闭
```

**效果**：导出的HTML支持图片查看和缩放！

## ❓ 常见问题

**Q：格式检测不准确？**  
A：确保光标在文字内部，点击几次或按方向键。

**Q：图片查看器在编辑器中能用吗？**  
A：只在导出的HTML文件中可用。

**Q：链接插入后能修改吗？**  
A：选中链接，删除后重新插入。
