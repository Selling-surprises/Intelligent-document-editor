# 功能演示指南

## 🎯 快速体验所有功能

### 1️⃣ 界面预览

打开应用后，您会看到：
- **顶部**：蓝色标题栏显示"离线Word文档编辑器 v3.8"
- **工具栏**：包含所有编辑工具的按钮栏
- **中央**：白色A4纸张大小的编辑区域，带有优雅的阴影
- **左侧**：目录面板（桌面端，有标题时显示）
- **底部**：状态栏显示字符数和字数

### 2️⃣ 文本编辑演示

#### 步骤1：输入文本
```
点击白色编辑区域，输入：
"这是一个功能强大的文档编辑器"
```

#### 步骤2：应用格式
1. 选中"功能强大"
2. 点击工具栏的 **B**（加粗）按钮
3. 结果：**功能强大**

#### 步骤3：添加颜色
1. 选中"文档编辑器"
2. 点击调色板图标（🎨）
3. 选择蓝色
4. 结果：文字变成蓝色

### 3️⃣ 标题和目录演示

#### 创建文档结构
```
输入以下内容（每行后按回车）：

第一章 项目介绍
第二章 功能特性
第三章 使用说明
```

#### 设置标题
1. 选中"第一章 项目介绍"
2. 点击"标题"下拉菜单
3. 选择"标题 1"
4. 对其他章节重复操作

#### 查看目录
- 左侧会自动显示目录面板
- 点击目录项可快速跳转

### 4️⃣ 颜色功能演示

#### 文本颜色
```
输入文本："重要提示"
```
1. 选中文本
2. 点击调色板图标
3. 选择红色
4. 文字变成红色

#### 背景高亮
```
输入文本："关键信息"
```
1. 选中文本
2. 点击荧光笔图标
3. 选择黄色
4. 文字添加黄色背景

### 5️⃣ 链接插入演示

#### 插入网页链接
1. 点击链接图标（🔗）
2. 在对话框中输入：
   - 链接地址：`https://www.example.com`
   - 显示文本：`访问示例网站`
3. 点击"插入链接"
4. 结果：显示蓝色下划线的"访问示例网站"
5. 点击链接会在新标签页打开

#### 快速链接（不输入显示文本）
1. 点击链接图标
2. 只输入链接地址：`https://github.com`
3. 显示文本留空
4. 点击"插入链接"
5. 结果：直接显示URL作为链接文本

### 6️⃣ 列表功能演示

#### 无序列表
```
输入：
功能特点
```
1. 点击无序列表图标
2. 输入第一项："易于使用"
3. 按回车继续输入
4. 输入第二项："功能丰富"
5. 按回车继续输入
6. 输入第三项："完全免费"

#### 有序列表
```
输入：
使用步骤
```
1. 点击有序列表图标
2. 输入："打开编辑器"
3. 按回车，输入："开始编辑"
4. 按回车，输入："导出文档"

### 7️⃣ 图片插入演示

1. 点击图片图标
2. 选择一张本地图片
3. 图片自动插入到光标位置
4. 图片自动适应文档宽度

### 8️⃣ 撤销和重做演示

1. 输入一些文字
2. 点击撤销按钮（↶）
3. 文字消失
4. 点击重做按钮（↷）
5. 文字重新出现

### 9️⃣ 设置功能演示

#### 打开设置面板
1. 点击右上角的设置图标（⚙️）
2. 设置面板从右侧滑出

#### 自定义标题
1. 在"网页标题"输入框输入："我的文档"
2. 浏览器标签页标题立即更新

#### 调整透明度
1. 拖动"内容区域透明度"滑块
2. 观察编辑区域透明度变化
3. 可以设置0-100%之间的任意值

#### 设置背景图片
1. 点击"背景图片"下的文件选择按钮
2. 选择一张图片
3. 编辑区域会显示背景图片
4. 配合透明度调整可以看到更好的效果

### 🔟 导出文档演示

1. 编辑完成后，打开设置面板
2. 点击"导出为HTML文件"按钮
3. 文件自动下载
4. 用浏览器打开导出的HTML文件
5. 所有格式、样式、图片都完整保留

## 🎨 完整示例文档

试试创建这样一个文档：

```
我的项目计划

第一章 项目概述
这是一个重要的项目，旨在提升用户体验。

主要目标：
• 提高效率
• 降低成本
• 优化流程

第二章 实施步骤
1. 需求分析
2. 方案设计
3. 开发实施
4. 测试验证

第三章 相关资源
更多信息请访问：[项目官网](https://example.com)

注意事项
请特别关注安全性和性能问题。
```

应用格式：
- "我的项目计划" → 标题 1
- "第一章 项目概述" → 标题 2
- "第二章 实施步骤" → 标题 2
- "第三章 相关资源" → 标题 2
- "注意事项" → 标题 3
- "重要" → 红色文字
- "安全性和性能" → 黄色高亮

## 💡 使用技巧

### 快速格式化
1. 先输入所有内容
2. 再统一设置格式
3. 最后调整细节

### 颜色搭配建议
- **红色**：警告、重要提示
- **黄色高亮**：关键信息
- **蓝色**：链接、标题
- **绿色**：成功、完成

### 文档结构建议
```
标题 1 → 文档标题
标题 2 → 章节标题
标题 3 → 小节标题
正文 → 普通内容
```

### 导出前检查
- ✅ 检查所有链接是否正确
- ✅ 确认图片都已插入
- ✅ 预览目录结构
- ✅ 检查字数统计
