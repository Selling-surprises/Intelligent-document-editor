# 如何使用目录功能

## 📖 目录功能简介

离线Word文档编辑器支持自动生成目录功能。当您在文档中添加标题后，导出的HTML文件会自动包含一个美观的悬浮目录。

---

## ✅ 第一步：添加标题

### 什么是标题？

标题是文档的章节标记，用于组织文档结构。标题有三个级别：

- **标题 1（H1）**：用于章节，如"第一章"、"引言"
- **标题 2（H2）**：用于小节，如"1.1 背景"、"1.2 目的"
- **标题 3（H3）**：用于子小节，如"1.1.1 研究现状"

### 如何添加标题？

#### 方法1：先输入文字，再设置为标题

1. 在编辑器中输入文字，例如："第一章"
2. 选中这段文字（用鼠标拖动选择）
3. 点击工具栏中的**"标题"下拉菜单**
4. 选择"标题 1"、"标题 2"或"标题 3"

#### 方法2：先设置标题，再输入文字

1. 将光标放在要添加标题的位置
2. 点击工具栏中的**"标题"下拉菜单**
3. 选择"标题 1"、"标题 2"或"标题 3"
4. 输入标题文字

### 如何确认标题添加成功？

标题添加成功后，您会看到：

✅ **文字变大**：标题文字比正文大
- 标题 1 最大
- 标题 2 中等
- 标题 3 较小

✅ **文字变色**：标题文字变成蓝色

✅ **左侧目录显示**：左侧的悬浮目录会自动显示该标题

### 示例

假设您要创建以下文档结构：

```
第一章 引言
  1.1 研究背景
  1.2 研究目的
第二章 方法
  2.1 实验设计
  2.2 数据收集
```

操作步骤：

1. 输入"第一章 引言"，选中，设置为"标题 1"
2. 输入"1.1 研究背景"，选中，设置为"标题 2"
3. 输入"1.2 研究目的"，选中，设置为"标题 2"
4. 输入"第二章 方法"，选中，设置为"标题 1"
5. 输入"2.1 实验设计"，选中，设置为"标题 2"
6. 输入"2.2 数据收集"，选中，设置为"标题 2"

## 📤 第二步：导出文档

### 如何导出？

1. 点击右上角的**"设置"按钮**（齿轮图标）
2. 在设置面板中，点击**"导出HTML"按钮**
3. 浏览器会自动下载HTML文件

### 导出成功提示

导出成功后，会显示提示信息：

- 如果有标题：显示"文档已导出！目录已生成（X个标题）。打开HTML文件后，将鼠标移到左侧边缘或点击蓝色'☰'按钮查看目录。"
- 如果没有标题：显示"文档已导出！提示：添加H1-H3标题可自动生成悬浮目录。"

---

## 👀 第三步：查看目录

### 打开导出的HTML文件

1. 找到下载的HTML文件（通常在"下载"文件夹）
2. 双击打开（会在浏览器中打开）

### 显示目录的三种方法

#### 方法1：鼠标悬停（推荐）

1. 将鼠标移动到页面的**最左侧边缘**
2. 目录会自动从左侧滑出
3. 鼠标移开后，目录会自动隐藏

**提示**：这是最自然的方式，就像翻开书的目录页一样。

#### 方法2：点击切换按钮

1. 在页面左侧中间位置，找到**蓝色的"☰"按钮**
2. 点击按钮，目录会固定显示
3. 再次点击按钮，目录会隐藏

**提示**：按钮有蓝色光晕闪烁，很容易找到。

#### 方法3：使用浏览器开发者工具（高级）

1. 按 **F12** 键打开开发者工具
2. 切换到 **Elements** 标签
3. 按 **Ctrl+F**（或 **Cmd+F**）搜索 `floating-toc`
4. 右键点击找到的元素 → **Scroll into view**

### 使用目录

#### 点击目录项

1. 点击目录中的任意标题
2. 页面会平滑滚动到对应的位置
3. 标题会短暂高亮显示（黄色背景）

#### 目录层级

目录会显示标题的层级关系：

- **一级标题（H1）**：不缩进，字体正常
- **二级标题（H2）**：缩进15px，字体稍小
- **三级标题（H3）**：缩进30px，字体更小

## 🔧 常见问题

### Q1：我添加了标题，但导出后没有目录

**可能原因**：

1. **没有正确添加标题**
   - 只是把文字加粗或放大，不是标题
   - 必须使用工具栏的"标题"下拉菜单

2. **浏览器缓存问题**
   - 浏览器可能缓存了旧版本的HTML文件

**解决方法**：

1. **检查是否正确添加标题**
   - 标题文字应该变大且变成蓝色
   - 左侧悬浮目录应该显示该标题

2. **查看控制台信息**
   - 导出时按 **F12** 打开控制台
   - 查看"找到的标题数量"是否大于0
   - 如果是0，说明没有正确添加标题

3. **清除浏览器缓存**
   - 按 **Ctrl+Shift+Delete**（或 **Cmd+Shift+Delete**）
   - 选择"清除缓存"
   - 重新打开HTML文件

### Q2：导出的HTML文件有目录，但看不到

**可能原因**：

目录默认隐藏在页面左侧边缘外，需要手动显示。

**解决方法**：

1. **方法1**：将鼠标移到页面最左侧边缘
2. **方法2**：查找并点击蓝色的"☰"按钮
3. **方法3**：按F12，搜索 `toc-toggle`，右键 → Scroll into view

### Q3：我看不到蓝色的"☰"按钮

**可能原因**：

1. 浏览器窗口太小
2. 按钮被遮挡
3. 没有仔细查看

**解决方法**：

1. **最大化浏览器窗口**
2. **滚动到页面中间位置**
3. **仔细查看页面最左侧边缘**
4. **按钮特征**：
   - 蓝色背景
   - 白色"☰"图标
   - 有蓝色光晕闪烁动画
   - 位于页面左侧中间位置

### Q4：点击目录项没有反应

**可能原因**：

1. JavaScript被浏览器阻止
2. HTML文件损坏

**解决方法**：

1. **检查浏览器设置**
   - 确保允许运行JavaScript
   - 某些浏览器默认阻止本地HTML文件的JavaScript

2. **查看控制台错误**
   - 按 **F12** 打开控制台
   - 查看是否有错误信息（红色文字）

3. **重新导出**
   - 返回编辑器
   - 重新导出HTML文件

### Q5：如何让目录默认显示？

**方法1：修改HTML文件（简单）**

1. 用文本编辑器打开HTML文件
2. 搜索 `translateX(-240px)`
3. 将其改为 `translateX(0)`
4. 保存文件

**方法2：修改CSS（推荐）**

1. 用文本编辑器打开HTML文件
2. 找到 `.floating-toc` 的CSS样式
3. 将 `transform: translateY(-50%) translateX(-240px);` 改为 `transform: translateY(-50%) translateX(0);`
4. 保存文件

## 💡 使用技巧

### 技巧1：合理使用标题层级

**好的标题结构**：
```
第一章 引言          （H1）
  1.1 研究背景       （H2）
    1.1.1 国内现状   （H3）
    1.1.2 国外现状   （H3）
  1.2 研究目的       （H2）
第二章 方法          （H1）
  2.1 实验设计       （H2）
  2.2 数据收集       （H2）
```

**不好的标题结构**：
```
标题1               （H1）
标题2               （H1）
标题3               （H1）
```

### 技巧2：使用有意义的标题文字

**好的标题**：
- "第一章 引言"
- "1.1 研究背景"
- "实验方法"

**不好的标题**：
- "标题1"
- "章节"
- "内容"

### 技巧3：定期保存和导出

1. 编辑过程中定期导出，避免数据丢失
2. 导出的HTML文件可以在任何浏览器中打开
3. 可以将HTML文件分享给他人

### 技巧4：使用目录快速导航

1. 长文档使用目录可以快速跳转
2. 固定显示目录，方便频繁查看
3. 目录层级清晰，一目了然

## 📱 移动端使用

### 在手机或平板上查看

1. 将HTML文件传输到移动设备
2. 用浏览器打开
3. 点击蓝色"☰"按钮显示目录
4. 目录会自动适配小屏幕

### 移动端特点

- 目录宽度自动调整
- 触摸操作支持
- 平滑滚动效果
- 自动隐藏，节省空间

---

## 🎨 目录外观

### 视觉设计

- **背景**：半透明白色，不遮挡内容
- **文字**：清晰易读，层级分明
- **按钮**：蓝色主题，有光晕动画
- **动画**：平滑的滑入滑出效果

### 交互反馈

- **鼠标悬停**：目录项变蓝色，向右移动
- **点击**：页面平滑滚动，标题高亮
- **切换按钮**：悬停时放大，点击时切换状态
