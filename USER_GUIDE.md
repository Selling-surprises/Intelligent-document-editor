## 📝 段落首行缩进功能

### 使用方法

1. **选中段落**：点击或选中要设置的段落文字
2. **打开段落对话框**：
   - 方式1：右键点击 → 选择"段落" → "段落设置"
   - 方式2：使用工具栏的段落按钮
3. **设置首行缩进**：
   - 在"缩进和间距"选项卡中
   - 找到"特殊格式"下拉框
   - 选择"首行缩进"
   - 输入缩进值（如：2字符）
4. **应用设置**：点击"应用"按钮

### 效果说明

- ✅ **水平缩进**：第一行文字向右缩进指定字符数
- ✅ **垂直对齐**：段落垂直位置保持不变，不会向上或向下偏移
- ✅ **后续行正常**：第二行及以后的行不受影响

### 示例

```
设置前：
这是一段测试文字。这是一段测试文字。这是一段测试文字。
这是一段测试文字。这是一段测试文字。

设置首行缩进2字符后：
    这是一段测试文字。这是一段测试文字。这是一段测试文字。
这是一段测试文字。这是一段测试文字。
```

## 📊 表格边框样式功能

### 10种边框样式

| 样式名称 | 效果说明 | 使用场景 |
|---------|---------|---------|
| 🚫 **无框线** | 移除所有边框 | 制作无边框表格、数据展示 |
| ⬜ **所有框线** | 表格和所有单元格都有边框 | 标准表格、数据表 |
| 🔲 **外侧框线** | 仅表格外边框 | 简洁表格、内容分组 |
| ⊞ **内部框线** | 仅单元格之间的边框 | 分隔单元格内容 |
| ☰ **内部横框线** | 仅水平方向的内部边框 | 行分隔、列表样式 |
| ⫴ **内部竖框线** | 仅垂直方向的内部边框 | 列分隔、对比表 |
| ⎺ **上框线** | 仅表格顶部边框 | 标题下划线、顶部强调 |
| ⎽ **下框线** | 仅表格底部边框 | 表格底部分隔、总计行 |
| ⎸ **左框线** | 仅表格左侧边框 | 左侧强调、引用样式 |
| ⎹ **右框线** | 仅表格右侧边框 | 右侧强调、注释样式 |

### 使用方法

#### 步骤1：选中表格
- 右键点击表格的任意位置

#### 步骤2：打开表格属性
- 在右键菜单中选择"表格" → "表格属性"

#### 步骤3：切换到边框选项卡
- 点击对话框顶部的"边框"标签

#### 步骤4：选择边框样式类型
- 在"边框样式类型"区域
- 点击10种样式中的任意一种
- 选中的样式会高亮显示

#### 步骤5：调整边框参数（可选）
- **边框宽度**：拖动滑块调整（0-10px）
- **边框线型**：选择实线、虚线、点线或双线
- **边框颜色**：使用颜色选择器或输入颜色代码

#### 步骤6：应用设置
- 点击对话框底部的"应用"按钮
- 表格边框立即生效

### 实用案例

#### 案例1：制作简洁的数据表
```
样式：外侧框线
边框宽度：2px
边框线型：实线
边框颜色：#333333

效果：表格有外边框，内部无边框，看起来简洁大方
```

#### 案例2：制作行分隔列表
```
样式：内部横框线
边框宽度：1px
边框线型：虚线
边框颜色：#cccccc

效果：每行之间有虚线分隔，适合列表展示
```

#### 案例3：制作标题表格
```
样式：下框线
边框宽度：2px
边框线型：双线
边框颜色：#000000

效果：仅底部有双线边框，适合标题或总结行
```

## 🎨 界面优化

### 工具栏简化

我们移除了以下按钮，让工具栏更加简洁：

- ❌ **全屏编辑按钮**：可以使用浏览器的全屏功能（F11）
- ❌ **打印按钮**：可以使用浏览器的打印功能（Ctrl+P 或 Cmd+P）

### 替代方案

| 移除的功能 | 替代方案 | 快捷键 |
|-----------|---------|--------|
| 全屏编辑 | 浏览器全屏 | F11 (Windows/Linux)<br>Ctrl+Cmd+F (Mac) |
| 打印 | 浏览器打印 | Ctrl+P (Windows/Linux)<br>Cmd+P (Mac) |

## 💡 使用技巧

### 技巧1：快速设置表格边框
1. 创建表格后，立即右键 → 表格属性
2. 选择"所有框线"作为默认样式
3. 根据需要调整边框宽度和颜色

### 技巧2：段落首行缩进的最佳实践
- **中文文档**：建议使用2字符缩进
- **英文文档**：建议使用4-5字符缩进
- **代码文档**：建议使用4字符缩进

### 技巧3：表格边框组合使用
虽然当前不支持边框样式组合，但可以通过以下方式实现类似效果：
1. 先设置"外侧框线"
2. 然后手动调整单元格样式（通过浏览器开发者工具）

## 🐛 已知问题和限制

### 限制1：表格边框应用范围
- **当前**：边框样式应用于整个表格
- **未来**：计划支持单个单元格的边框设置

### 限制2：边框样式组合
- **当前**：只能选择一种边框样式
- **未来**：计划支持多种样式组合（如同时设置上框线和下框线）
