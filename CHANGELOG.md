```typescript
const handleMouseDown = (e: MouseEvent) => {
  if (selectionMode) {
    // 按住Ctrl键：添加/移除单元格
    if (e.ctrlKey || e.metaKey) {
      const cellIndex = selectedCells.indexOf(cell);
      if (cellIndex > -1) {
        // 已选中，移除
        const newCells = selectedCells.filter((_, i) => i !== cellIndex);
        setSelectedCells(newCells);
      } else {
        // 未选中，添加
        const newCells = [...selectedCells, cell];
        setSelectedCells(newCells);
      }
    } else {
      // 不按Ctrl：替换选区
      setSelectedCells([cell]);
    }
  }
};
```

## v3.14.1 (2025-12-06)

### 🐛 紧急Bug修复

#### 🔧 修复表格选择闪烁问题
- ✅ **问题**：鼠标拖动选择表格时，单元格一直处于闪烁状态，无法正常选中多个单元格
- ✅ **原因**：
  - `handleMouseMove`事件在鼠标移动时频繁触发
  - 即使鼠标在同一个单元格内移动，也会不断调用`setSelectedCells`和`highlightCells`
  - 导致频繁的状态更新和DOM操作，造成视觉上的闪烁
- ✅ **解决方案**：
  - 添加`currentEndCell` ref来记录当前的结束单元格
  - 在`handleMouseMove`中检查结束单元格是否变化
  - 只有当鼠标移动到不同的单元格时才更新选区
  - 避免不必要的状态更新和DOM操作

### 🔍 技术改进

#### 优化前的代码（有问题）
```typescript
const handleMouseMove = (e: MouseEvent) => {
  if (!isSelecting.current || !startCell.current) return;

  const cell = target.closest('td, th') as HTMLTableCellElement;
  
  if (cell && editor.contains(cell)) {
    // ❌ 每次mousemove都会执行，即使在同一个单元格内
    const cells = getCellsInRange(startCell.current, cell);
    setSelectedCells(cells);  // ❌ 频繁更新状态
    highlightCells(cells);    // ❌ 频繁操作DOM
  }
};
```

**问题分析**：
- 鼠标在单元格内移动时，`mousemove`事件会触发多次
- 每次都会调用`getCellsInRange`、`setSelectedCells`和`highlightCells`
- 即使选区没有变化，也会重复执行
- 导致React频繁重新渲染，DOM频繁更新
- 视觉上表现为闪烁

#### 优化后的代码（已修复）
```typescript
// 添加ref记录当前结束单元格
const currentEndCell = useRef<HTMLTableCellElement | null>(null);

const handleMouseDown = (e: MouseEvent) => {
  // ...
  currentEndCell.current = cell; // ✅ 初始化
};

const handleMouseMove = (e: MouseEvent) => {
  if (!isSelecting.current || !startCell.current) return;

  const cell = target.closest('td, th') as HTMLTableCellElement;
  
  if (cell && editor.contains(cell)) {
    // ✅ 检查是否是同一个单元格
    if (cell === currentEndCell.current) return;
    
    // ✅ 更新结束单元格
    currentEndCell.current = cell;
    
    // ✅ 只在单元格变化时才更新
    const cells = getCellsInRange(startCell.current, cell);
    setSelectedCells(cells);
    highlightCells(cells);
  }
};

const handleMouseUp = () => {
  // ...
  currentEndCell.current = null; // ✅ 重置
};
```

**改进效果**：
- ✅ 只在鼠标移动到不同单元格时才更新
- ✅ 避免不必要的状态更新
- ✅ 避免不必要的DOM操作
- ✅ 消除闪烁现象
- ✅ 提升性能

// 获取矩形选区内的所有单元格
const getCellsInRange = (start: HTMLTableCellElement, end: HTMLTableCellElement) => {
  const startRow = (start.parentElement as HTMLTableRowElement).rowIndex;
  const endRow = (end.parentElement as HTMLTableRowElement).rowIndex;
  const startCol = start.cellIndex;
  const endCol = end.cellIndex;

  const minRow = Math.min(startRow, endRow);
  const maxRow = Math.max(startRow, endRow);
  const minCol = Math.min(startCol, endCol);
  const maxCol = Math.max(startCol, endCol);

  const cells: HTMLTableCellElement[] = [];
  for (let i = minRow; i <= maxRow; i++) {
    for (let j = minCol; j <= maxCol; j++) {
      cells.push(table.rows[i].cells[j]);
    }
  }
  return cells;
};
