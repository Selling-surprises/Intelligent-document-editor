# 链接卡片功能演示

## 🎯 功能亮点

### 1. 双模式链接插入
- **文本链接**：传统的超链接样式
- **卡片链接**：现代化的卡片展示

### 2. 灵活的卡片配置
- 必填：链接地址、卡片标题
- 可选：卡片描述、卡片图片

### 3. 精美的视觉效果
- 圆角边框设计
- 悬停动画效果
- 响应式布局

## 💡 使用技巧

### 技巧1：选择合适的链接类型

**使用文本链接的情况**：
- 文章中的参考链接
- 内联引用
- 不需要特别强调的链接

**使用卡片链接的情况**：
- 推荐资源
- 重要链接
- 需要展示预览信息的链接

### 技巧2：优化卡片内容

**标题**：
- 简洁明了（建议10-20字）
- 突出重点
- 避免过长

**描述**：
- 补充说明（建议50-100字）
- 提供关键信息
- 吸引用户点击

**图片**：
- 使用高质量图片
- 建议尺寸：240x240px以上
- 使用HTTPS协议

### 技巧3：合理布局

**单个卡片**：
- 适合重点推荐
- 占据较大空间
- 视觉冲击力强

**多个卡片**：
- 适合列表展示
- 保持间距一致
- 避免过于密集

### 技巧4：配色建议

**卡片背景**：
- 使用白色或浅色
- 与页面背景形成对比
- 保持清晰可读

**边框颜色**：
- 默认：浅灰色
- 悬停：主题色（蓝色）
- 保持一致性

## 🔧 高级用法

### 用法1：自定义样式

虽然当前版本不支持直接自定义样式，但可以通过以下方式实现：

1. 导出HTML文件
2. 在`<style>`标签中修改`.link-card`样式
3. 调整颜色、大小、间距等

### 用法2：批量插入

如果需要插入多个卡片：

1. 准备好所有卡片信息（Excel或文本文件）
2. 逐个插入卡片
3. 保持格式一致

### 用法3：卡片组合

可以将卡片与其他元素组合使用：

```
标题
正文段落...

相关资源：
[卡片1]
[卡片2]
[卡片3]

继续正文...
```
